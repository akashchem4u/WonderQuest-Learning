Reading Load Rules (K–1)
Max question text: 15 words. Emoji accent encouraged but not required on every question.
Answer labels: ≤ 3 words each. Single word strongly preferred.
Phoneme display: Show phoneme chunks on all sight-word / CVC questions. Max 3 chunks per word.
No grade labels in UI. World names only (Ocean World, Dream World, etc.).
Wrong answer: Never say "wrong". Use "Oops!" or "Let's try again!". Stars are never deducted in session.
Hint: Always available via button tap. Never auto-shown for first attempt. Auto-shown only after 2 wrong picks.

Hint System Rules
Trigger: Manual (💡 Hint button) or auto after 2 wrong picks in same question.
Hint format: Coach avatar + 1–2 sentence text + optional visual (phoneme split, counting dots, pattern preview).
No penalty for using hint. Hint flag sent in POST /session/answer for analytics only.
XP on hinted correct: Full XP awarded. Never penalize for hint use in child UI.
