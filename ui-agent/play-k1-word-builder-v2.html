Component Identity

Fileplay-k1-word-builder-v2.html
AudienceK–1 (ages 5–7)
SkillPhonics / spelling
Width375px mobile
FontNunito 900
Accent#9b72ff violet

Interaction Model

Input methodTap-to-place (no drag required)
Slot behaviorActive slot advances after each tap
Wrong letterSlot shakes, resets to empty, no star loss
UndoTap filled slot to clear it
Word length3–4 letters (CVC pattern focus)

Rules

No timers. No grade labels on any screen. Hint shows first letter only after 2nd wrong attempt. Replay never deducts stars. Coach voice reads word prompt on mount. Used tiles dim (opacity 30%) after placement. Celebration: full word spelled out with bounce animation.
