Ask Your School Admin
Your IT coordinator or department admin can look up your class code in the WonderQuest school portal.
What to tell your admin:
1️⃣You need your WonderQuest teacher class code
2️⃣Your name and subject/grade level

📋 Message template for your admin
"Hi, I'm trying to access WonderQuest for my [Grade X / Subject] class but my class code isn't working. Could you check the Admin Portal and send me the correct code? It should start with WQ-. Thank you!"
