  • 375px mobile · Nunito 900 · various accent colors per band
  • Compact ribbon always visible at top of play screens

Ribbon Elements

  • Star count — left, colored to match accent
  • Progress dots — 8 dots, filled = done, current = glowing, empty = locked
  • World indicator — right, world emoji + name
  • Bottom band — 3px stripe in accent color

Tab Variants

  • P0 Gold #ffd166: Meadow world · Sight Words questions
  • P1 Violet #9b72ff: Ocean world · Phonics questions
  • P2 Mint #58e8c1: Forest world · Vocabulary questions
  • P3 Coral #ff7b6b: Volcano world · Comprehension questions

Design Rules

  • Ribbon height: 38px (compact, never distracting)
  • NO timers, NO grade labels shown in ribbon
  • Progress dots never show "wrong" — only done/current/not-yet
