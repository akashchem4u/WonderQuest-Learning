Component Spec: Intervention Plan Editor v2
  • Desktop 900px+, Sora, bg #f0f4f8, nav #2a5080
  • Two-panel layout: 280px learner card left, fluid plan editor right
  • Learner card: avatar, name, tier badge, key metrics, mini skill history bars
  • Plan editor: goal textarea, date pickers, frequency select, strategy chip multi-select
  • Success criteria: checkbox list (visual only, state not persisted in HTML)
  • Resource links: icon + label list with add-link input
  • Tab 2: strategy library chip grid grouped by category (Numeracy, Engagement, Literacy)
  • Tab 3: timeline Gantt-style bars per week, milestone markers
  • Tab 4: team share with toggle per member; parent contact via WQ messaging only (contact-safe)
  • Save Draft / Save & Share actions in header
  • Contact-safe: no teacher or parent PII exchanged; all communication routes through WQ
