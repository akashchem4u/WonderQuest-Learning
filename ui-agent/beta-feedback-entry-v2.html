Severity Guide
Use this to calibrate your severity rating
When in doubt

Rate one level higher than you think. It's easier for the team to downgrade than to miss a critical issue.

Low — Minor Polish
Cosmetic issue, typo, minor visual misalignment. Doesn't affect usability. Example: button label truncated by 1 character.
Medium — Noticeable Friction
Feature works but with confusion or extra steps. User can recover. Example: badge popup requires 3 taps to dismiss.
High — Impaired Experience
Core feature broken or unreliable. User frustrated, task incomplete. Example: streak saver never shows when streak breaks.
Blocker — Session-Ending
App crashes, infinite loop, data loss, or user cannot proceed at all. Requires immediate fix. Example: app freezes on badge screen every time.
