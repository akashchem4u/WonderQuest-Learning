Tab 5 — Spec: API routes, KPI definitions, formulas, design tokens.
All other tabs consume these definitions. This tab is reference-only.
API Routes
GET /owner/adoption/summary — dashboard KPIs, health score, band summary, alerts
GET /owner/adoption/bands — per-band adoption data, classroom table; query param: ?band=P0|P1|P2|P3
GET /owner/adoption/teachers — uptake tiers, at-risk list, top teachers
GET /owner/adoption/trends — 8-week readiness scores, component scores, milestone status
KPI Definitions
Active Household = ≥1 child session in last 7 days
Active Teacher = ≥1 assignment or review action in last 7 days
Band Adoption % = (active households in band ÷ total enrolled households in band) × 100
Daily Active Children = unique child IDs with ≥1 session on a given calendar day (rolling 7-day avg)
Teacher Uptake Tiers
Power = 10+ sessions per week
Regular = 3–9 sessions per week
Light = 1–2 sessions per week
Inactive = 0 sessions per week
At-risk threshold: teacher inactive ≥14 days gets outreach flag
Readiness Score Formula
Score = weighted avg of component scores:
P0 Feedback ×0.30 + Route Health ×0.25 + Content Coverage ×0.25 + QA Pass Rate ×0.20
Reference: same formula as owner-release-readiness-v2.html
Score range 0–100. Threshold: ≥85 = Launch-Ready. Current: 81 → Near-ready.
