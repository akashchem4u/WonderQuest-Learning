Child Safety Rules
World names only: Never show grade labels (G2-3, K-1). Use Dream/Ocean/Fire/Forest etc.
Node names: Creative names (Wave Riders, Coral Reef). Never show skill IDs.
Locked world tap: Shows friendly "Next World?" prompt — never "You're not ready".
Max worlds shown: Current band's worlds + 1 teaser. Do not show full curriculum map.
