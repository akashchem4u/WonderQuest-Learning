Content guidelines for teachers
✅ OK to include: Progress updates, activity assignments, positive observations, schedule reminders, general classroom updates
❌ Do not include: Grades or test scores, diagnostic labels, comparison to other students, personally identifiable information, external contact info
⚠️ Route through admin: Serious concerns, disciplinary matters, medical or SPED topics — use the admin channel, not direct messages

Contact-Safe Rules
No PII in URLs: Teacher messages route through /messages/threads/:id — no parent email or phone in any URL param or query string.
In-app only: All messages delivered through WonderQuest app. No external email gateway for teacher→parent.
Teacher sees: Parent display name + child name only. No email, phone, or address.
Content filter: Client-side soft block on grade/test score language. Hard block on personal contact info patterns (email/phone regex).
Message retention: 12 months then archived. FERPA-compliant access log.
Layout
Desktop (≥ 768px): 2-col — 300px list / 1fr thread.
Mobile (<768px): Single-col — list view → thread view (back nav). Thread fills screen.
