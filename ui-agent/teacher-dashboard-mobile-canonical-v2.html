Engineering · Spec Reference — teacher-dashboard-mobile-canonical-v2
This is the true mobile home, not a reduced desktop. Designed for between-class use.

// ROUTES

RoutePurpose
/teacher/homeMobile canonical home
/teacher/rosterFull student roster
/teacher/student/:idIndividual student context
GET /teacher/:id/dashboard-summaryKPIs, SNS counts, top-3 support
POST /teacher/noteAdd student/parent note
POST /teacher/messageMulti-parent message

// SNS TIER DEFINITIONS

TierLeft-borderConditionAction priority
🚩 Flag#9a2a2aSupport tier ≥ 2 consecutive weeksHighest — show first
👁 Watch#c07800Support tier 1 weekMedium — show second
💪 Strong#1a8050Strength tier current weekLow — shown last

// QUICK ACTION RAIL

ActionTap behaviorContext-aware?
Add noteBottom sheet, pre-filled with last-viewed studentYes
Assign activityBottom sheet, student picker + activity selectorYes
Full rosterRoute to /teacher/rosterNo
Class reportRoute to /teacher/class-summaryNo
Message parentsMulti-select sheet, flag+watch pre-selectedAuto-selects

// DESIGN TOKENS

TokenValueUsage
teacher-bg#f0f4f8Screen background (cool steel)
teacher-nav#2a5080Hero bar, primary CTA
teacher-fontSoraAll text
flag-border#9a2a2aLeft border on flag rows
watch-border#c07800Left border on watch rows
strong-border#1a8050Left border on strong rows
today-divider#f0d070 on #2a5080TODAY chip in intervention log

// KEY DIFFERENCES FROM DESKTOP

FeatureMobile canonicalDesktop
LayoutSingle column, vertical scroll3-col (student / detail / notes)
Student listTop 3 flag/watch on home; full roster on tapAlways-visible left lane
Note entryBottom sheetInline right-panel form
Skill detailFull screen after tapCenter panel inline
Filter controlsHorizontal scroll chipsFilter bar + sort dropdown
