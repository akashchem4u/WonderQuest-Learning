"Which of these is a living thing?" — sci.life.animals P0
QA check rules. Each script must pass all three checks before approval.
🔒
Check 1: Doesn't Reveal Answer

The script must not contain the correct answer, point explicitly to the correct option, or use phrases like "the answer is..." or "remember that X is...". The script should prompt thinking without guiding toward the answer.

Fail example: "There are 3 apples. Three is the right number!"
Pass example: "Count the apples carefully. How many do you see?"
👶
Check 2: Age-Appropriate Language

Scripts are for learners aged 4–8. Avoid: complex sentence structures, words beyond Grade 2 vocabulary, jargon, or abstract concepts. Max sentence length ~8 words for P0–P1.

Fail example: "Consider the taxonomic classification..."
Pass example: "Which animal lives in the water?"
🗣️
Check 3: Pronunciation Guide

Any phonics-related script must include a phonetic pronunciation note for the target sound. Use IPA notation (e.g. /b/ not "buh"). This ensures TTS and voice actors pronounce consistently.

Warn example: "buh sound" (informal)
Pass example: "/b/ as in ball" (IPA-informed)
