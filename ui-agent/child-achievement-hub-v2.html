TAB 5 — Engineering Spec: Child Achievement Hub v2

Routes
GET /player/achievements → star total, badge count, recent achievements (last 3), nav counts
GET /player/badges → array of badge objects: {id, emoji, name, world, state, earnedAt, description}
GET /player/trophies → trophy groups: {name, icon, badges[], totalBadges, earnedBadges}
GET /player/worlds → worlds: {name, emoji, state (complete|current|locked), nodesComplete, totalNodes, starsEarned, badgesEarned[]}
GET /player/streak → {currentStreak, weekDots[], history (28 days), nextMilestone, daysToNextMilestone}

Achievement Hub Sections
• Badges (shelf + detail): grid of 52px circles, 4-col; tap earned → detail sheet with name/desc/date/share
• Trophies (group progress): compact rows with progress bar, mini badge row, % complete chip
• World Unlocks: 5 world cards vertical; tap complete world → trophy overlay with all earned badges
• Streak + Replay: flame hero + week dots + 28-day calendar + locked-badge replay CTA cards

Badge States
• earned: gold border + glow-pulse animation — celebration state
• new: mint border + NEW chip — earned within last 3 days (earnedAt > now - 3 days)
• locked: grayscale + dashed border + 🔒 overlay — "Coming Soon" label only, no gating language

Trophy Group Completion
Completion % = count(earned badges in group) / count(total badges in group) × 100
Progress bar: gold fill for earned portion; violet fill variant for incomplete groups

Streak History
Show exactly 28 days (4 rows × 7 cols calendar grid)
Filled dot = played that day; empty dot = neutral (no label, no negative framing, no "missed")
Today's dot = outlined gold with glow; week row shows M T W T F S S labels

Replay CTA Logic
Show 2–3 locked badges; each badge card shows: badge emoji + badge name + world that unlocks it + "Play Now! 🚀" button + "+N ⭐ possible" stars chip
Source: GET /player/badges?state=locked — pick first 3 with defined unlock world

World Overlay (complete world tap)
Sheet modal: all earned badges in that world (grid), star total earned in world, "Replay for stars 🌟" CTA

Child Safety Rules
• No grade labels anywhere in the UI
• No skill IDs exposed to child
• No shame framing: locked = "Coming soon! ✨", empty streak dots = no label
• Locked states use curiosity framing, never deficit framing
• World names only: Dream World 🌙, Ocean World 🌊, Fire World 🔥, Star World ⭐, Crystal World 💎
