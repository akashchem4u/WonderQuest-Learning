Language Rules

  • Never: "error", "connection failed", "network unavailable", "timeout"
  • Always: friendly, animals/rockets/space metaphors
  • Reassure: stars and badges are safe
  • "Saved Games" instead of "Cached Content"

States

  • Offline — dinosaur floats gently, dots animate, Try Again button
  • Connecting — rocket pulse ring, fun fact to entertain while waiting
  • Back Online — celebration burst, resume exactly where they left off
  • Cached — pre-downloaded content playable offline

Animation

Float animation on hero icon (3s cycle). Dot blink animation for waiting state. Pulse ring for connecting. No jarring transitions.
