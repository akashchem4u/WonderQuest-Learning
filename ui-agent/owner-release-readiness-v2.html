📋 Release Readiness Spec
Routes / API
EndpointPurpose
GET /release/readinessFull readiness report: score, check results, blockers
GET /release/blockersActive blockers (P0/P1 feedback + route incidents)

POST /release/holdPlace release on hold with reason
GO Criteria (all must pass)
P0 feedback: Zero open P0 feedback items.
Route health: All core routes OK (p95 < target, err < 0.5%). Zero degraded routes.
Content coverage: All active nodes ≥ 6 questions. Warnings allowed (accept-for-release); criticals (<4) block.
QA score: ≥ 80% all active skills. No hard-block failures in active pool.
Stakeholder sign-off: All required sign-offs recorded.
Readiness score: ≥ 90/100 required for automatic GO.
