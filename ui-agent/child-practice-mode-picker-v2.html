Mode Picker spec. Two-mode chooser — zero skill/difficulty labels.

Design Rules

  • No skill labels, no difficulty labels, no band names
  • Emoji-first, big friendly cards
  • Quest = story, journey, stars
  • Challenge = speed, gold, streak
  • Random pick option for indecisive learners

Quest vs Challenge

Quest follows the world map linearly. Challenge is a daily 5-question burst from any skill. Both reward XP but Challenge gives gold bonuses.

Random Pick

Selects between Quest (continue current node) and Challenge (if not done today) with a fun spin animation.
