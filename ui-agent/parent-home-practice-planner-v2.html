Spec 🛠️
Parent Home Practice Planner v2
API Routes
GET /parent/practice-plan?childId=:id&week=YYYY-WNN
Returns the weekly practice plan with 3 activities (1 support, 1 growing, 1 strength). Auto-generated from SNS tier data.
GET /parent/practice-suggestions?childId=:id&week=YYYY-WNN
Returns world-themed suggestions + optional teacher suggestions. Refreshes weekly.
POST /parent/reminders
Body: { childId, time, days[], activityIds[], enabled }. Creates or updates reminder schedule.
GET /parent/practice-history?childId=:id&limit=N
Returns parent self-reported practice history. Includes emoji sentiment and childNote fields.
Plan Generation Logic
Auto-generated from child's SNS tier data each week. Always 3 activities: 1 support area + 1 growing strength + 1 strength. Tier-to-activity mapping: support tier → activity targeting that skill; strength → celebratory or extension activity. Ties broken by recency of skill update.
Tier Labels (Parent Language)
Support area — amber accent. Forward-looking: "building confidence here!"
Growing strength — teal accent. Celebratory of progress.
Strength — green accent + 🌟. Celebratory, extension focus.
NEVER use: "weak", "failing", "behind", accuracy %, raw scores, missed-day framing.
Reminder Channels
Primary: push notification. Email optional. Respects parent.settings.quietHours — server validates delivery window before dispatch. Days stored as ISO weekday array (1=Mon…7=Sun). Time stored as HH:MM UTC-offset aware.
Practice History
Parent self-reported only — NOT linked to in-app session data. Fields: activityId, date, status (saved|completed), sentiment (great|okay|tricky), childNote (freeform). Streak calculated server-side: weeks with ≥1 completed entry.
World-Themed Suggestions
Activity ideas tied to child's current WonderQuest world for engagement. World pulled from child.currentWorld. Suggestion bank tagged by world + skill. Teacher suggestions sourced from teacher.classSuggestions[] — optional, shown only when array is non-empty.
Parent Language Rules (STRICT)
NEVER: Show accuracy %, raw scores, missed-day framing, "weak", "failing", "behind".
ALWAYS: Forward-looking, growth-focused language. Activities should feel like play, not schoolwork. Warm, encouraging, family-safe tone throughout.
Design Tokens

Phone frame: 375px, border-radius:40px, warm parchment bg
