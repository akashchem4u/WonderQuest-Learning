Teacher Queue Empty States spec. Four distinct empty/low-data states with actionable guidance.

States

  • Queue Empty (All Clear): celebration + proactive suggestions while queue is clear
  • First-Time Setup: guided checklist for new teachers
  • Waiting for Data: class set up, students not yet played — gentle waiting state
  • Low Activity: students enrolled but inactive — nudge tools

Design Principles

  • Empty state is never blank — always provide value or a next step
  • Celebrate the positive (all clear = great work!)
  • Don't alarm when things are just quiet — frame as normal
  • Always give teacher a clear action they can take
