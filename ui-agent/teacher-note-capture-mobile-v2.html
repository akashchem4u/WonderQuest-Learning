Spec
Component Spec: Note Capture Mobile v2
  • 390px mobile, Sora, bg #f0f4f8, nav #2a5080
  • Tab 1: quick entry — student autocomplete, 4 note type chips, textarea, quick tags, Save + Add Another
  • Note types: Observation (blue), Concern (red), Achievement (green), Parent Contact (amber)
  • Quick tags: pre-set clickable chips; multi-select; toggle on/off
  • Character counter shown below textarea (current/max)
  • Tab 2: review screen before save — shows formatted card, confirms append-only log behavior
  • Tab 3: recent notes list with type badge, student name, 2-line preview, tags
  • Tab 4: note templates by category — tap to pre-fill Quick Entry
  • Bottom CTA: Save Note (primary) + Add Another (secondary green)
  • Contact-safe: all parent communications route through WQ only; no PII shown
  • Once saved, notes append to intervention log and cannot be edited or deleted
