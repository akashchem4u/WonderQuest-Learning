Component Spec: Owner Adoption Detail Desktop v2
  • Desktop 900px+, Sora, bg #0d1117, gold #ffd166
  • KPI strip: Overall Adoption, Active Households, Avg Sessions/Week, Week 4 Retention
  • Tab 1 overview: band adoption summary table + SVG polyline trend per band
  • Tab 2: full band detail table with completion % mini-bar per row; P3 alert callout
  • Tab 3: school uptake horizontal SVG bar chart — green ≥80%, amber 50–79%, red <50%
  • Tab 4: retention curve — 4 circles (W1–W4) with % and drop; cohort comparison bar chart
  • Trend arrows: ↑ green, → flat grey, ↓ red; inline with table rows
  • All charts use inline SVG — no external library required
  • Export CSV button in nav right; data window toggle: 7-day / 30-day
  • Alert callout for declining bands/schools: red border card with action recommendation
