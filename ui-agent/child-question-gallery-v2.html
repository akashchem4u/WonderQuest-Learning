Engineering · Spec Reference — child-question-gallery-v2
Browse and replay surface only. All question IDs are opaque — never show skill names to child.

// ROUTES

RoutePurpose
/child/:id/galleryGallery home (current world default)
/child/:id/gallery/:questionIdIndividual question detail/replay
GET /child/:id/questions?worldId=:wIdAll questions for this world (unlocked only)

// QUESTION TILE STATES

StateVisualInteraction
Completed (3 stars)Gold star row, teal border glowTap to replay
Completed (1–2 stars)Partial stars, muted borderTap to replay (improve stars)
Current (playable)Violet glow border, NEW chipTap → quest map for play
LockedLock icon, 0.4 opacity, dashed borderToast: "Finish more quests to unlock!" — never modal

RuleDetail
