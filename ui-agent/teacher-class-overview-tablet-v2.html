TAB 5 · Engineering Specification Tablet frame: 768×1024px (iPad portrait). Layout system, design tokens, routes, and notes for implementors below.

Frame & Layout

Device frame768px × 1024px, iPad portrait, border-radius: 20px
Canvas bgwhite (#ffffff)
2-col layoutgrid-template-columns: 280px 1fr — used in Overview + Queue tabs
Card gridsgrid-template-columns: 1fr 1fr — used in Assignments + At-Risk tabs
Touch targetsMinimum 44px height / interactive area (tablet standard)
Tab barBottom of frame, 5 tabs, active = navy bg with gold top border
Top nav height54px, sticky, navy #2a5080

API Routes (same as desktop teacher)

  • GET /api/teacher/class/:id/overview — Class overview, SNS counts, engagement
  • GET /api/teacher/class/:id/support-queue — Prioritized Flag + Watch list
  • GET /api/teacher/class/:id/assignments — All assignments with completion data
  • GET /api/teacher/class/:id/at-risk — At-risk students with history
  • POST /api/teacher/interventions — Create note / action
  • POST /api/teacher/messages — Send parent message
  • POST /api/teacher/assignments/create — Create new assignment
  • PATCH /api/teacher/assignments/:id/resend — Resend to incomplete

Tab Behaviors

  • Tab 1 (Overview): 2-col layout; left = class summary; right = chart + wins + band table; full-width quick-actions strip at bottom.
  • Tab 2 (Support Queue): 2-col; left = queue list with filter chips; right = selected student detail panel. Empty state shown until row tapped.
  • Tab 3 (Assignments): Filter chips → active/scheduled/closed; 2-col card grid; resend button visible only when completion < 60%.
  • Tab 4 (At-Risk): 2-col card grid; tap card → inline expand below card with 8-week SNS dots + intervention log (not a sidebar).
  • Tab 5 (Spec): This view — engineering reference only.

Tablet vs Desktop vs Mobile

  • Desktop (1280px+): 3-col layout (left sidebar / main / right sidebar)
  • Tablet (768px): 2-col (280px fixed / 1fr), no right sidebar; queue detail in right col
  • Mobile (390px): Single col; queue detail as bottom drawer; no chart bar labels
