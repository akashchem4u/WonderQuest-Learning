Rules

Focus mode is set per-profile (not per-session). Auto-TTS fires on question load. All child safety rules still apply: no timers, no shame, replay-safe. Parent/caregiver can toggle from settings at any time. Reduce motion collapses all CSS animations to instant transitions.
