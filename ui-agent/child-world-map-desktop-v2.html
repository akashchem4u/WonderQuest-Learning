child-world-map-desktop-v2 — Specification

Persona: Child · Desktop 900px+ · Nunito 900 font · #100b2e bg · #9b72ff primary violet

API: GET /child/worlds returns [{ id, name, emoji, state, nodes_done, nodes_total }]

API: GET /child/world/{id}/nodes returns node list with state per node

  • World names: friendly only (Dream World, Ocean World, Fire World, Star World) — NEVER grade/level numbers
  • Locked world tap: friendly "Coming soon — keep playing to unlock!" — NEVER "not ready" or negative framing
  • States: locked (dim, no pointer) / unlocked (violet border) / complete (mint border + Done chip) / active (gold glow)
  • Node states: done (mint dot) / current (violet glow dot) / next (outlined) / locked (dim)
  • World complete state: stars display, collect badge CTA, go to next world CTA
  • No timers, no grade labels, no accuracy %, no shame language anywhere
  • Desktop 900px+ grid: 4-col world grid, scrollable if more worlds added
