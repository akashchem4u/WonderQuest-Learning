Overlay Stacking — Simultaneous Rewards

If multiple reward events fire at the same time (e.g. badge + level-up + streak on the same question):

Sequence order (never stack): 1. Badge Earned 2. Level Up 3. Streak Saved 4. Node Unlocked 5. Session Complete

Queue overlays in an array. When one dismisses (auto or tap), dequeue and show next. Session Complete always shows last (it is the summary of everything).
