Media Hint Library spec. Browse, edit, and manage visual/audio hints attached to questions.

Hint Types

  • Emoji — single Unicode emoji character
  • Image — PNG/JPG up to 2MB, must have alt text
  • Audio — MP3/OGG up to 1MB, pronunciation guide or sound clip

Deletion Rules

Hints used in 1+ questions cannot be deleted without first removing associations. The system warns if deletion would leave questions without a hint.

Skill Association

Each hint is associated with one skill. A hint may appear on questions from other skills but the primary association determines filter placement.
