# WonderQuest Tools

This folder contains local generation and deployment helper scripts for the
WonderQuest build.

- `build_planning_workbook.py`: generates the first planning workbook from the launch JSON files
- `check_render_deploy_ready.sh`: validates Render-ready config, required env vars, and a local production build
- `render_post_setup_check.sh`: validates the live Render URL after deployment and checks key routes/gates

Use this folder for build-time and deployment helper scripts only.
