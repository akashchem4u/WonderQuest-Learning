# PWA Icons

Generate the following icons and place them in this directory:

- `icon-192.png` — 192x192 px (used for home screen and manifest)
- `icon-512.png` — 512x512 px (used for splash screen and manifest)

Both icons should use `purpose: "any maskable"` and follow the
[maskable icon safe zone](https://web.dev/maskable-icon/) guidelines
(keep the logo within the inner 80% circle).

Tools to generate icons:
- https://maskable.app/editor
- https://realfavicongenerator.net/
- https://pwa-asset-generator
