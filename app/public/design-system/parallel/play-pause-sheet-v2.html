Trigger Sources

  • X button in topbar (always visible during play)
  • Back gesture (Android / swipe right on iOS)
  • Home button / app backgrounding (auto-pause)
  • Parent tap on overlay (via passcode flow)
  • On trigger: session state saved to persistent storage immediately
  • ✅ Auto-save fires BEFORE sheet opens

Star-Safe Guarantee

  • Stars earned before pause are ALWAYS kept
  • Even if app is force-quit after pause — stars already credited
  • Progress saved chip shows: "You've earned ⭐ N stars so far"
  • Quit confirm also shows star-safe message
  • ❌ Never say stars could be lost if they leave
  • ❌ Never guilt-trip: "Are you sure you want to give up?"

Sheet Dimensions

  • Child pause: ~40% of frame height
  • Parent pause: ~55% (more controls)
  • Quit confirm: ~45% (confirm card is taller)
  • Scrim: rgba(8,5,20,0.60) + 2px blur
  • Sheet: border-radius: 24px 24px 0 0
  • Drag handle: cosmetic only in v2

Button Priority

  • "Resume" is always primary (most prominent)
  • "Leave" is always danger / muted styling (intentional)
  • "Stay and play!" in quit confirm is primary (pushes toward staying)
  • "Leave" in quit confirm is secondary red (smaller, left)
  • Parent access: ghost/small — not tempting to child
  • ❌ Never make "Leave" the primary button
