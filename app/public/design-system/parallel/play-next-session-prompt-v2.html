Trigger Rules

  • Same Day Bonus: sessions_today === 1 (first complete of day)
  • Daily Limit: sessions_today >= max_sessions_per_day
  • Default max: 3 sessions/day (parent override in settings)
  • Tomorrow Teaser: always shown when child chooses "See you tomorrow" OR hits limit
  • ❌ Never show "Want to keep going?" after daily limit reached

Language Rules

  • ✅ "Want to keep going?" — curious, not pushy
  • ✅ "See you tomorrow!" — warm, no guilt
  • ✅ "Your adventure goal!" — frames limit positively
  • ❌ Never "You can't play anymore today"
  • ❌ Never "You've reached your limit"
  • ❌ Never countdown to next available session

Bonus Star

  • Same-day bonus: +1 ⭐ awarded at START of 2nd session
  • Gold chip: "+1 Bonus for playing again today!"
  • Not shown on 3rd session (only 2nd session gets bonus)
  • Not shown if child already at daily limit
  • ❌ Never show bonus as a reason to play (enticement rule)

Tomorrow Teaser Content

  • Teaser: next skill in curriculum sequence
  • Copy: "Tomorrow you'll explore [skill]!" + 1 example
  • Icon: subject-specific emoji (📚 reading, 🔢 math, etc.)
  • If next skill unknown: generic "more adventures await!"
  • Star-safe message always shown at close
  • No skill revealed if parent has "surprise mode" enabled
