ENG NOTE — teacher-intervention-timeline-v3.html
Standalone intervention timeline component — used in the full-history view of a student's intervention record.
NOT the same as the timeline inside intervention-detail: This is a richer, dedicated view reached from "View full history" link on intervention detail page.
Timeline event types: trigger (queue flag created) | teacher_note (freetext note logged) | mastery_transition (status change) | system_adjustment (difficulty auto-adjusted) | resolution.
Mastery transitions shown: just status changes (Just started → Building → Strong). Raw score changes NOT shown in timeline (only in detail page).
Privacy: Teacher notes shown in full (private to teacher). No per-student accuracy %, no wrong answers, no parent info.

Retention

Resolved interventions are archived and visible for 90 days. After 90 days, intervention record is anonymised (student name replaced with "Student [hash]") but statistics are retained for class analytics. Teacher notes are deleted at 90 days.
