Database Schema
-- Asset library asset_library ( id TEXT PRIMARY KEY, type TEXT CHECK(type='audio'), voice_session_id TEXT, -- session ref only; no name stored locale TEXT, -- en-GB / en-US / en-AU linked_skill_id TEXT, status TEXT, -- pending | needs_rerecord | approved | blocked uploaded_at TIMESTAMPTZ ) -- QA metrics (auto-populated by pipeline) aud_qa_metrics ( asset_id TEXT REFERENCES asset_library(id), lufs NUMERIC, peak_db NUMERIC, noise_floor_db NUMERIC, leading_silence_ms INTEGER, trailing_silence_ms INTEGER, transcript_match_pct NUMERIC, checked_at TIMESTAMPTZ, PRIMARY KEY (asset_id, checked_at) ) -- Whisper ASR transcripts aud_transcripts ( asset_id TEXT REFERENCES asset_library(id), transcript_json JSONB, -- word-level timestamps locale TEXT, generated_at TIMESTAMPTZ, PRIMARY KEY (asset_id, locale) )
ASR Pipeline
Model Whisper medium
Timestamps Word-level timestamps stored in transcript_json
Phoneme comparison Post-ASR via phoneme alignment tool; IPA diff computed per segment
Match threshold >90% required to pass; <85% triggers needs_rerecord auto-status
Re-record Trigger Logic
Trigger conditions LUFS out of range OR transcript match <85%
Auto-status set needs_rerecord
Notification Content team notified via review_queue event
Voice actor privacy Session reference ID only stored; no name persisted. Field: voice_session_id TEXT in asset_library. Contractual privacy requirement.
