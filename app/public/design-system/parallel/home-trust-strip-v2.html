Database schema

trust_certifications ( id SERIAL PRIMARY KEY, badge_key VARCHAR(64) UNIQUE NOT NULL, -- e.g. "coppa", "ferpa", "no_ads" label VARCHAR(128) NOT NULL, -- display label icon_emoji VARCHAR(8) NOT NULL, short_desc TEXT NOT NULL, -- 1–2 sentence summary long_desc TEXT NOT NULL, -- full explanation for modal cert_url VARCHAR(512), -- link to policy / cert page valid_until DATE, -- null = perpetual is_active BOOLEAN NOT NULL DEFAULT TRUE );

Expiry alert trigger

🔒

Interaction & privacy rules

Click tracking
No click events stored. Trust strip is purely informational — no analytics to comply with GDPR and our own data-minimization commitment.
Modal opens
Client-side only; no server request on badge click
Content updates
Fetched from GET /api/trust-certifications?active=true at page load; cached 24 h in CDN
Language / tone
Positive, reassuring language only. No fear-based framing.
