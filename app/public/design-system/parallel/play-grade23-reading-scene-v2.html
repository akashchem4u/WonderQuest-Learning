📋 play-grade23-reading-scene-v2 · Engineering Spec

Purpose

  • G2–3 reading comprehension scene — short 3–5 sentence passage + 1 question with 4 answer choices
  • Tests literal comprehension (detail recall, sequence, cause-effect) at G2–3 level
  • Mint theme. No timer, no accuracy %, stars never decrease.

Tabs / States

  • Active Passage card with key sentence highlighted, question, 4 choice buttons (full sentence each)
  • Correct Passage border → green, key sentence gets stronger green highlight, correct choice → green, mint CTA
  • Wrong Wrong choice shakes + red, banner points back to highlighted passage section
  • Spec This document

Passage Card

  • 3–5 sentences, 0.92rem Nunito 700, line-height 1.65 for readability
  • Key sentence (the answer-containing sentence) has .key-sentence class → subtle mint background highlight
  • On correct: .correct-hl added → green background highlight on key sentence, card border turns green
  • On wrong: same highlight stays visible — directs re-read attention

Answer Choices

  • 4 full-sentence choices (not single words) — reflects real reading comprehension format
  • Staggered slide-in animation (translateX from -12px, not scale — text reads better with horizontal entry)
  • Wrong: shake + red border, other choices remain active for retry
  • Correct: green border, others dimmed

Bottom Bar — Reading-Specific

  • "📖 Re-read" replaces "🔊 Replay" — text-first reading scene (no audio dependency at G2-3)
  • Re-read tapping scrolls passage into view (if off-screen) and flashes highlight
  • 💡 Hint: coach points to the highlighted sentence ("The answer is in the highlighted part!")
  • Star-safe badge on active + wrong states

Passage Design Principles

  • Character-driven narrative preferred over abstract expository text
  • Question types: What/When/Where literal recall, simple cause-effect ("Why did Maya…?"), sequence ("What happened after…?")
  • Distractor answers: one tempting confusion (parks → park was mentioned), two clear distractors
  • Answer is always explicitly stated in passage — no inference required at G2-3 literal level

Design Tokens (G2–3 Mint)

  • Mint: #58e8c1 — progress segs, passage title, key sentence highlight, CTA
  • Key sentence active: rgba(88,232,193,0.15) bg
  • Key sentence correct: rgba(80,232,144,0.2) bg
  • CTA: linear-gradient(135deg, #58e8c1, #30b090), text #0a2a22
