ENG NOTE — parent-practice-planner-v2.html
In-app practice schedule planner for the parent. Sets: practice days per week, session count per day, time-of-day preference, reminder on/off.
These settings feed the WonderQuest algorithm — the child's session scheduling suggestions on the play home screen are based on parent's plan.
No timer — session duration is NOT set here. Sessions end naturally when child stops or hits the session limit count.
COPPA note: Reminder pushes can only be sent to parent's device — never to child's device directly.
Settings persist per child_id. Shown in: parent-family-dashboard Settings tab and parent-link-child-flow Step 3 (goals).

Algorithm Integration

preferred_time_of_day feeds the play home screen's "Good time to play!" nudge for the child. The child's home screen can show a gentle "Afternoon is your best time ⭐" message if the parent has set an afternoon preference and it's currently afternoon.

practice_days feeds the streak calculation — a day where no session was expected (e.g. Saturday when weekends are off) does not break the streak.
