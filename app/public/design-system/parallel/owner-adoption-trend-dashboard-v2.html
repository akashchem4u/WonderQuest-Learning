Access Control

  • Owner only gate: is_owner=true — not accessible to teachers, parents, or content team
  • All adoption metrics are aggregate; no individual student or family data is exposed

Data Source & Freshness

  • Source: Analytics warehouse (daily snapshots) · 4–6 hour data lag · not real-time
  • Table: adoption_snapshots(date, mau, dau, schools_count, families_count, dau_mau_ratio) refreshed daily
  • Activation funnel events sourced from analytics_events table — each funnel step = a distinct event type

Metric Definitions

  • MAU — Users with ≥1 session in the calendar month
  • DAU/MAU ratio — Stickiness metric · Healthy threshold: >20% · Concerning: <15%
  • Activation funnel — Each step represents a distinct event logged in analytics_events
  • School data — School names shown in B2B contract context · Individual student names are NEVER shown

Export Options

  • CSV export — aggregate metrics only; no PII included
  • PDF export — board and investor use; formatted summary with KPI row and trend chart

School Engagement Classification

  • HighDAU/MAU ≥ 70%
  • MediumDAU/MAU 40–69%
  • LowDAU/MAU < 40%
