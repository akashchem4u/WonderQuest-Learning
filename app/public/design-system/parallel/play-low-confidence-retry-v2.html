Confidence Model

  • Low confidence: skill wrong 3+ times across 2+ sessions
  • Confidence score: 0–100, based on correctness history
  • Low (<30%): offer practice at session start
  • Medium (30–60%): optional practice, not shown unless requested
  • High (>60%): no practice offered
  • Practice improves score: +10% per correct answer

Practice Questions

  • Always 3 questions (not adjustable in v2)
  • Scaffolded: Q1=2 choices, Q2=4 choices, Q3=4 choices (harder)
  • Phoneme/picture hints always shown (full scaffolding)
  • Stars: +1 ⭐ per correct answer (same as regular)
  • No stars lost on wrong practice answers
  • ❌ Never 5 practice Qs — 3 max to keep it quick

Opt-In Rules

  • Practice is ALWAYS opt-in — child can skip
  • "Skip to regular session" always visible
  • No guilt language: "Skip to regular session" not "Skip practice"
  • If skipped 3+ times: stop offering for that skill
  • Practice shown max once per session-start, not mid-session

Confidence Meter

  • Height: 10px (skill review) / 12px (done screen)
  • Low: gradient #ff7b6b → #ffd166
  • Medium: gradient #ffd166 → #9b72ff
  • High: gradient #58e8c1 → #9b72ff
  • Done screen shows before/after comparison
  • ❌ Never show % number to child — just the bar
