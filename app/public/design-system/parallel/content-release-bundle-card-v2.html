Release Bundle Cards

Content bundles staged for owner publish — grouped by release version
ENG NOTE — Release Bundle Card v2: A release bundle = a named set of skills to be published together in a single deploy.
Bundles staged by curriculum team → owner sees summary + can approve/publish from owner console.
Bundle status: Draft → Curriculum Ready → Owner Review → Published (or Rejected → back to Curriculum Ready).
Publish gate: ALL skills in bundle must have readiness_score ≥ 90 AND review_status = Approved before owner can publish bundle.
Publishing a bundle sets delivery_locked=false on all included skills and creates a governance_log entry.
Partial publish: owner may cherry-pick skills from a bundle (creates partial publish record; remaining skills stay in bundle).
