Variants: Simple (icon+value+label), With Delta (adds ▲▼ % change), With Sparkline (mini 7-day line chart), With Bar (value + progress/fill bar), With Comparison (current vs prev period). Audience-aware: Owner sees financial+operational KPIs (MRR/MAU/uptime); Teacher sees classroom KPIs; Parent sees child progress metrics. Never show financial KPIs to teacher/parent.

MetricCardProps Interface

interface MetricCardProps { // Required label: string value: string | number size: 'compact' | 'standard' | 'wide' audience: 'owner' | 'teacher' | 'parent' state: 'loaded' | 'loading' | 'error' | 'empty' // Optional base sublabel?: string icon?: string // emoji or icon name // Variant: Delta delta?: { value: number unit: '%' | 'pts' | 'abs' label: string // e.g. "vs last month" direction: 'up' | 'down' | 'neutral' } deltaPositiveDirection?: 'up' | 'down' // 'down' for churn (falling=good) // Variant: Sparkline sparkline?: number[] // 7 values, one per day // Variant: Bar bar?: { value: number max: number } // Variant: Comparison comparison?: { current: number previous: number label: string // e.g. "vs last week" } }

Size Definitions

Size Min Width Grid Fit Best For
compact ~220px 4-column grid Quick KPIs, summary stats, owner dashboard strips
standard ~280px 3-column grid Delta cards, comparison, teacher dashboards
wide ~400px 2-column grid Sparkline charts, parent progress, detailed breakdowns

Variants

Variant Trigger Prop(s) Description
Simple — (base) Icon + large value + label + sublabel. Default when no variant props provided.
With Delta delta Adds ▲▼ directional chip below value. Color determined by direction + deltaPositiveDirection rule.
With Sparkline sparkline Renders a 7-bar mini chart. Highest bar and current day bar highlighted in full accent green.
With Bar bar Renders fill bar below value. Color: green ≥80%, amber ≥50%, red <50%.
With Comparison comparison Shows current vs previous period rows + mini side-by-side bar visual.

State Behaviors

State Rendering
loaded Normal card. All variant props rendered as specified.
loading Shimmer skeleton replaces value, sublabel, icon, and variant areas. Card structure preserved.
empty Value shows — or 0 in muted color. No delta shown. Optional italic sublabel "No data yet".
error Red ! icon, "Unable to load" text, subtle red border tint, "Try again" retry link.

Delta Color Logic

Condition Chip Color Example
direction=up + deltaPositiveDirection=up (default) green MAU up 8.3% — good
direction=down + deltaPositiveDirection=up (default) red MAU down — bad
direction=down + deltaPositiveDirection=down green Churn down 0.4pts — good
direction=up + deltaPositiveDirection=down red Churn up — bad
direction=neutral amber Uptime unchanged

Audience Visibility Rules

KPI Category Owner Teacher Parent
Financial (MRR, ARR, churn, revenue) visible hidden hidden
Operational (uptime, MAU, schools) visible hidden hidden
Class aggregate (participation, avg session) hidden visible hidden
Student performance per-student accuracy % hidden hidden hidden
Child progress (XP, streak, mastered) hidden hidden visible
Negative framing ("missed day", accuracy %) — — never shown
