📋 play-prereader-picture-word-scene-v3 · Engineering Spec

Purpose

  • Pre-K picture-to-word matching scene — child sees a picture (emoji) and must tap the word that names it
  • Bridge between visual recognition and early reading — supports pre-literacy sight-word exposure
  • Gold (#ffd166) Pre-K theme throughout — same as counting/letter/sound/shapes scenes
  • No timer, no accuracy %, stars never decrease

Tabs / States

  • Active Picture hero with gold glow, 2×2 word grid (word + distractor emoji hint), bottom 3 buttons
  • Correct Hero → green, correct word btn gets selected-correct, others get dimmed, gold CTA "Next Word ✨ +1⭐"
  • Wrong Wrong word btn gets selected-wrong + shake, red banner with actionable guidance, star-safe shown
  • Spec This document

Picture Hero

  • 160×160px card, border-radius: 20px, 3px gold border, letter-glow animation
  • Large emoji (72px) centered in card — no text in hero, text is in the word choices
  • On correct: border → #50e890, background → #0d2a1a, correct-glow keyframe fires once

Word Choice Buttons

  • 2×2 grid, Nunito 900, 1.5rem word text — large enough for pre-readers to trace letters
  • Small emoji hint below word text (subtle, 0.9rem, 50% opacity) — scaffolding for hesitant readers
  • obj-pop staggered entry animation (0.05s per button)
  • Hover: gold border + -2px lift
  • Wrong: selected-wrong (red border + shake) — other choices remain active for retry
  • Correct: selected-correct (green border), others dimmed (0.35 opacity, pointer-events none)

Coach Script Examples

  • Active: "Duh-duh-dog! Which word says what this picture is?"
  • Correct: "Yes! D-O-G spells dog! You matched the picture to its word — that's reading!"
  • Wrong feedback banner: "Not quite — look at the picture. Say the name out loud, then find the right word!"

Word Set Examples

  • CVC words only (3-letter consonant-vowel-consonant): cat, dog, sun, hat, bug, cup, bed, fox
  • One correct word + 3 phonetically distinct distractors (avoid rhyme traps at Pre-K level)
  • Emoji hint on distractor buttons gives context without making the wrong answer obvious

Progress Strip

  • Segment bars (not dots), flex:1 per question, 6px height, gold theme

Bottom Bar

  • Active/wrong: 🔊 Replay / 💡 Hint / 🤷 IDK yet (flex:1 each)
  • Star-safe badge below on active + wrong states
  • Correct: single gold CTA "Next Word ✨ +1⭐"

Design Tokens

  • Topbar: linear-gradient(135deg, #2a1800, #1a0c00)
  • Gold: #ffd166 — progress segs, hero border, CTA
  • CTA: linear-gradient(135deg, #ffd166, #f0a000), text #1a0c00
  • Correct: #50e890 / #0d2a1a bg
  • Wrong: #ff7b6b / #2a0e0e bg
  • Star-safe: #1a2a15 bg / #50e890 border
