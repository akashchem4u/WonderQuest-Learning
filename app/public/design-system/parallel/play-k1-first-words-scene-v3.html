📋 play-k1-first-words-scene-v3 · Engineering Spec

Purpose

  • K-1 sight/first-word reading scene — child reads a printed word and taps the matching picture
  • Higher-order than picture→word (Pre-K): child must decode the written word, not just recognize a shape
  • Violet (#9b72ff) K-1 theme. No timer, no accuracy %, stars never decrease.

Tabs / States

  • Active Large word hero (no picture cue), 2×2 answer cards with emoji + word label
  • Correct Word hero → green text, correct card highlighted, "You READ that word!" celebration
  • Wrong Wrong card shakes + red border, letter-by-letter sound-out prompt
  • Spec This document

Word Hero

  • 280×120px wide card (wider than tall — optimized for horizontal word display)
  • Nunito 900, 3.2rem — large enough to trace with finger
  • No picture cue — this tests word decoding, not guessing from visual context
  • violet-glow box-shadow animation while awaiting answer

Answer Cards

  • 2×2 grid, each card has large emoji + word label text below
  • Label text provided to support non-fluent readers (scaffolding)
  • obj-pop staggered entry animation
  • Wrong: shake + red border, other choices remain active
  • Correct: green border + dimmed others

Word Sets (K-1 Level)

  • CVC words: cat, dog, run, hop, jump, sit, big, red
  • High-frequency sight words: the, is, at, he, she, it, go, to
  • Action verbs preferred — more visually distractable than nouns

Coach Script Examples

  • Active: "Read the word — then tap its picture! Take your time — say each letter sound."
  • Correct: "You READ that word! J-U-M-P says 'jump' — and you knew it! That's real reading!"
  • Wrong: "Not quite — sound out each letter: j… u… m… p… What does it say?"

Design Tokens (K-1 Violet)

  • Violet: #9b72ff — progress segs, hero border, CTA
  • CTA: linear-gradient(135deg, #9b72ff, #7248e8)
  • Correct: #50e890 text + border / #0d2a1a bg
  • Star-safe badge always shown on active + wrong states
