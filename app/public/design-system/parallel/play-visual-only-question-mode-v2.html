Trigger & Content Rules

Rule Detail
Visual-only trigger child_audio_prefs.readout_enabled = false OR device audio unavailable
Content rule Every question MUST have visual_fallback_available = true — enforced at publish gate
Exception Phonics / rhyme skills marked audio_required = true — automatically excluded from session when audio is disabled
Adaptive exclusion When audio is off, audio_required=true questions excluded from adaptive selection pool:

WHERE audio_required = false OR audio_disabled = false
Image alt text alt_text required for all images (WCAG 1.1.1); enforced as publish blocker
Authoring rule Every image answer option must have descriptive alt text; emoji questions use the emoji character + name

Band Visual Design

Band P0 — Earliest Learners
  • Large images (120 px+)
  • 2-option layouts available
  • Maximum font size 20 px
  • High contrast imagery
Band P3 — Advanced
  • Richer text content
  • Images supplementary
  • Standard 4-option layout
  • Denser information display

Accessibility Standards (WCAG)

  • 1.1.1 Non-text content — alt text required for all images; enforced as a publish blocker
  • 1.3.3 Sensory characteristics — instructions never rely on shape, colour, or sound alone

CI Publish Gate Check

SQL — must return 0
SELECT COUNT(*)
FROM   content_questions
WHERE  has_image_asset = true
  AND  alt_text IS NULL
  AND  review_status = 'Published'

This check runs on every deploy. A non-zero result blocks the publish pipeline.
