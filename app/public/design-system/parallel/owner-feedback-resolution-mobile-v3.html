Component: owner-feedback-resolution-mobile-v3.html
Mobile post-resolution view. Same 3 states as desktop but single-column stacked. No sidebar. Meta merged inline. Reopen in sticky footer.

Layout Differences from Desktop

Sidebar meta merged into inline meta section. Activity log shown expanded by default on mobile (collapsed to last 3 entries on small phone heights). Reopen button in sticky footer action row (not sidebar). Reply textarea omitted on resolved/won't-fix items — reply was already sent.

Escalated Items on Mobile

Escalated items remain in the active queue on mobile — they are not "resolved" views. They appear here for reference alongside resolved/won't-fix items in the "All" filter. The mobile queue shows a 🚨 badge on the Feedback tab while any P0 escalation is open.
