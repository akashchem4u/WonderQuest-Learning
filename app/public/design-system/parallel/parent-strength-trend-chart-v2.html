Strength Definition
Condition mastery_status IN ('Growing','Strong') on ≥2 sessions in the period. NOT a test score.
Engagement Level Mapping
Exploring 1–2 sessions in the period
Growing 3–5 sessions with skill progression signals
Strong 6+ sessions with mastery signals
Privacy Rules
No accuracy %, no per-question data, no "Emma got this wrong" language anywhere in the UI.
Parent Language Rules
Always positive, forward-looking, adventure language. Use "revisited" not "repeated incorrectly". Use "exploring" not "attempting".
Database Schema
child_strength_trends ( child_id UUID, subject_id UUID, week_number INT, engagement_level ENUM('exploring','growing','strong'), sessions_count INT, updated_at TIMESTAMP ) -- Computed weekly by analytics job
Chart Implementation
Rendering SVG polylines drawn via JS; responsive width via viewBox scaling.
Reduced motion prefers-reduced-motion — removes line-draw animation, keeps static points visible.
Accessibility Chart has role="img" + aria-label text summary. Visually-hidden data table fallback via .sr-only.
