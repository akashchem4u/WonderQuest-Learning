File

Route trigger

States

Entry animation
modal-enter keyframe: scale 0.92→1 + translateY 12px→0, 0.3s cubic-bezier spring

Modal Structure

Header
Avatar (52px) + tag chip + title + ✕ dismiss
Progress section
World/node label, progress bar, node info dots
Last session strip
Time ago + session chip · bg #1a1540
Quest preview
Icon + quest name + meta + status badge
Star-safe badge
Always shown · green border · custom message per state
Actions
Primary (violet) → resume CTA · Secondary → home/map · Ghost danger → leave

Star-Safety Rules

Mid-quest interrupt
"Stars from your X answered questions are saved!" — partial stars kept
New quest state
Shows total stars from last session explicitly ("Your 21 stars are all here!")
Restart action
Label must say "keep stars" — stars never at risk from restart

Mid-Quest Progress Dots

Style
Horizontal row of 8 dots · done=green / active=violet pulse / pending=dim
Animation
pd-pulse: opacity 1→0.4 loop on active dot

Layout

Desktop
440px modal centered on backdrop · backdrop bg: rgba(10,8,32,0.85)
Mobile
Full-width bottom sheet (max 440px) · same content hierarchy
Z-index
Modal on top of all play UI · dismiss closes, does NOT navigate away
