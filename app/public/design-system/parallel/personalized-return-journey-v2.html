Voice, Theme, And Personalization

Personalized Return Journey

A re-entry sequence that changes shape based on how long the child has been away. Short gaps get a quick handoff back into play. Longer gaps get reassurance, favorite-theme comfort, and a staged confidence rebuild.

Data model

return_personalization_profiles
Stores gap band, favorite theme, confidence starter type, and current return phase.
return_journey_events
Append-only log of which steps were shown, completed, skipped, or suppressed.
return_voice_overrides
Optional temporary speech-rate overrides for comeback sessions.

Gap logic

2-day uses resume-first. 7-day uses favorite-theme plus warm-up. 30-day uses a full comeback path with fresh-start language and a stronger reassurance layer.

Copy rules

Never say the child fell behind, missed work, or broke a streak. Parent route can mention the operational return plan but should still avoid shame framing.

Success criteria

The first return session optimizes for completion, calmness, and confidence. Progression depth is secondary until the child shows steady re-engagement again.
