ENG NOTE — teacher-empty-class-state-v2.html
Teacher empty states — covers 3 distinct scenarios where class data is unavailable:
1. New account — no class set up yet. Teacher just created account, hasn't linked a class/rostered students.
2. Class exists but no activity today. Class is set up; students registered; just no sessions today.
3. No students in class. Class created, no students added yet (e.g. start of year).
Tone: Welcoming, practical. Not an error. Provide clear next steps.
Design tokens: Teacher persona — #f0f4f8 bg, #0f172a sidebar, #2563eb accent.

Tone Rules

All empty states are welcoming, not alarming. "No activity yet" states should never use: "No data", "Nothing here", "Error loading". Instead: "Good morning!" + yesterday's summary + tip about session timing. Progress from other days is always shown when available.
