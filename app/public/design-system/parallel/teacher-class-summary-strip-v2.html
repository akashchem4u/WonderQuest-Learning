Embeddable summary strip component — appears at the top of teacher home, command center, and analytics pages.
NOT a standalone page — this is a component specification file showing all strip variants.
Variants: Standard (5 stats) | Compact (icon row) | With alert pills | Mobile (horizontal scroll).
Stats shown: Active students / Total sessions / Class stars / Avg streak / Skills mastered.
Delta indicators: vs same period last week. ↑ green, ↓ amber (not red — no alarming language), = neutral.
Alert pills (optional): Only appear if queue_count > 0 or class milestone triggered. Pills: amber (attention) or blue/green (positive).

Component: teacher-class-summary-strip-v2.html
Usage: Embedded at top of teacher-home, teacher-command-center, teacher-analytics pages. NOT a standalone page.
Data source: /api/teacher/class-summary?period=week — returns stats for the currently selected time window.

Stats Definitions

StatDefinitionDelta baseline
Active studentsCount with ≥ 1 session in the periodSame period last week
SessionsTotal completed sessions (class)Same period last week
Class starsSum of all stars earned (class)Same period last week
Avg streakMean current_streak_days across all studentsSame period last week
Skills masteredCount of skill → "Strong" transitions this periodSame period last week

Delta Colour Rules

ChangeColourReason
Positive (↑)Green #22c55eGood signal
Negative (↓)Amber #f59e0bNeutral context — not alarming
Unchanged (=)Grey #aaaNo change

Down deltas are NEVER shown in red. Red is reserved for system errors and critical alerts only. A quieter week is normal — amber communicates "worth noting" not "alarming".

Alert Pills (With Alerts variant)

Pill colourWhen shown
AmberSupport queue has items / absence flags / hint pattern flags
BlueBand ceiling flags (positive — student ready to advance)
GreenClass milestone (e.g. N students mastered skill X)

Alert pills are optional — strip renders without them if no alerts are active. Pills are scrollable on narrow viewports.
