Accuracy Display Rule

Class-level aggregate accuracy % MAY be shown on class summary (e.g. "Class averaged 74% on Fractions this week"). Disclaimer mandatory: "Class average — individual student accuracy not displayed." Per-student accuracy NEVER shown anywhere in teacher portal.

Export

PDF export = 1-page summary: class name + week + stat row + band distribution + top skills + queue summary. No individual student names in export unless teacher explicitly requests per-student PDF (separate flow, FERPA consent required).
