digest_copy_bank Schema
-- Positive-only encouragement phrases for parent digests
CREATE TABLE digest_copy_bank (
  id          UUID         PRIMARY KEY DEFAULT gen_random_uuid(),
  persona     ENUM('parent', 'child')    NOT NULL,
  tone        ENUM('celebrate', 'encourage') NOT NULL,
  template    TEXT         NOT NULL,
  -- Example: "{{name}} had a fantastic week! {{emoji}}"
  locale      TEXT         NOT NULL DEFAULT 'en-US',
  created_at  TIMESTAMPTZ DEFAULT NOW()
);

-- Selection rule: avoid repeat from prior week
-- Query: pick random where id NOT IN (last_week_copy_id)
SELECT template FROM digest_copy_bank
WHERE  persona = 'parent'
  AND  id != last_used_copy_id
ORDER BY random() LIMIT 1;
Data Sources & Aggregation
play_sessions
Session count, total XP, subject breakdown for the 7-day window
skill_progress
Distinct subjects explored (for subject chips)
streak_records
Current streak length; milestone flag if streak is a multiple of 7
achievement_unlocks
Badge/achievement unlocks during the week (source for highlight moment)
badge_unlocks
Named badge unlocks (fallback highlight source)
Activity dots
One boolean per calendar day: EXISTS (SELECT 1 FROM play_sessions WHERE child_id = ? AND DATE(started_at) = day)
