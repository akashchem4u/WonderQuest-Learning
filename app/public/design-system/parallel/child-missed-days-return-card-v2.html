Component

File

Usage
Inline card at top of home hub on return after gap. Dismissible.
States
2-day gap (amber) · 7-day gap (mint) · 30-day comeback (gold)
Entry animation
card-enter: scale 0.94→1 + translateY 10px→0 · 0.4s spring

Language Rules

Never use
"Missed days", "you didn't play", "your streak is broken", "you fell behind"
2-day
"You've been away for a couple of days" — neutral, warm
7-day
"It's been a week!" — energetic, forward-looking
30-day
"Legend Returns!" / "It's been a whole month — this is huge!" — celebration, not shame

Week Bar

Played days
Orange flame dot (background #3a1800, border #f0a000)
Gap days
Dim dot with "·" — neutral gray, never red, label: "Away"
Today
Violet/gold border dot — "Today!" label

Star Safety

Badge
Always shown with explicit star count — "18 stars safe!"
30-day extra
"+Legend Badge earned!" on star-safe row. Gold color scheme.
Rule
Stars can never decrease from gap/absence. Shown explicitly to reassure.

Boost / Offer Section

7-day
Streak restore offer: "+1⭐ bonus for playing today". Forward-looking.
30-day
Comeback celebration: "+2⭐ +100XP". Legend Badge chip.
2-day
No boost offer — just warm re-engagement CTA.

Animations

mascot-wave: rotate -8→+8deg, 1.5s loop. Disabled on 30-day (static gold pose).
30-day shimmer
gold-pulse-shadow: box-shadow 16→32px rgba(255,209,102,0.15→0.35), 2s loop
