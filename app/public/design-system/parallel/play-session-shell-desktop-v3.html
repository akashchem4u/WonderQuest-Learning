States

Top Bar

Back button
← Home · triggers pause/save confirmation if mid-question
Progress strip
Dots: done=green / active=violet pulse / pending=dim. "Q4/8" label.
Star counter
Running total. Turns gold on correct answer momentarily.
Pause button
⏸ → opens play-pause-sheet modal

Question Zone

Image
160px rounded box · band-colored bg · center emoji for prototype
Answer grid
2×2 grid · hover=violet · correct=green · wrong=red + shake keyframe
No timer
Absolutely no countdown. No time pressure anywhere.

Right Rail

Mascot coach
mc-bounce animation · speech bubble · message changes per state
Progress card
Questions done/total · progress bar · star row (earned ⭐)
Star-safe badge
Always visible · message: "Stars safe — keep trying!" on wrong

Bottom Bar

Replay audio
Replays TTS question. Always available. Active state = violet border.
Hint
Progressive disclosure. Hint button turns violet after wrong answer.
I don't know yet
Red-tinted ghost button. Skips question, saves slot for retry. Stars safe.

Wrong Answer Rules

Language
"Not quite — give it another try!" / "That wasn't it" — never "Wrong!"
Shake anim
Wrong card: shake keyframe (translateX oscillation, 0.4s)
Stars safe
Stars do NOT decrease on wrong answer. Rail badge always present.
