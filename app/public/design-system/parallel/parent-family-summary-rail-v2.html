ENG NOTE — parent-family-summary-rail-v2.html
Horizontal scroll rail shown on the multi-child dashboard view (Family plan). One card per linked child.
Each card: avatar + name + band chip | mini stats (stars/sessions/streak) | top skill with mini bar | status dot + streak badge.
Tapping a card → switches active child context (same behaviour as parent-child-switcher-v3).
Add card pinned at end of rail. Adapts to desktop grid on wide viewports.
Privacy: No accuracy % in cards. Stars and session counts only.
