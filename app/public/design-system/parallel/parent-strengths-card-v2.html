Framing Rules

Stars shown = child's earned stars on that skill this week. Stars are child-language; parents understand stars = reward/progress. Never say "X% correct". Never say "mastery score". Use "Strong at X" and star counts.

If only 1 strength: add an encouraging note — "Building toward more strengths next week."

If 0 strengths: empty state with 🌱 icon. Copy: "Strengths appear after a few sessions in the same skill area." Never say "no strengths found" or anything negative.
