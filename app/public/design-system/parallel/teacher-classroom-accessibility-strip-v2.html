Spec — Teacher Classroom Accessibility Strip v2

Teacher can see

  • Student first names within their own class
  • Band assignment (P0–P3)
  • Relative activity level (visual bar only)
  • Active accessibility chips — icons only, no setting values
  • Total time in session
  • Aggregate accessibility counts per class

Teacher cannot see

  • Individual question accuracy
  • Which questions were attempted
  • Specific XP amounts
  • Accessibility setting values (e.g., which voice speed — just "slow reading" is on)
  • Student surnames or identifiers beyond first name
  • Parent-configured preference details

Privacy rules

  • Accessibility prefs configured by parents only
  • Teacher has read-only visibility of active features (awareness only)
  • Teacher cannot change a child's accessibility settings
  • Accessibility strip shows aggregate counts, never named individuals
  • Band grouping used in summary — never individual names

Classroom actions

  • Send positive encouragement banner to one or all students
  • Max 80 characters per message
  • Platform-routed only — not email or SMS
  • No targeting by accessibility status allowed
  • Encouragement is generic positive only — not evaluative

DB Query — Classroom Accessibility Summary

-- Classroom accessibility summary SELECT COUNT(*) FILTER (WHERE NOT readout_enabled) AS audio_off, COUNT(*) FILTER (WHERE voice_speed = 'slow') AS slow_reading, COUNT(*) FILTER (WHERE display_mode = 'high_contrast') AS high_contrast, COUNT(*) FILTER (WHERE reduced_motion = true) AS low_motion, COUNT(*) FILTER (WHERE large_buttons = true) AS large_buttons FROM child_accessibility_prefs cap JOIN class_memberships cm ON cm.child_id = cap.child_id WHERE cm.class_id = :class_id AND cm.is_active = true

FERPA Compliance

Aggregate counts only are exposed to the teacher. Individual accessibility preferences are accessible only to the parent who set them and school administrators. No personally identifiable education records are surfaced through this strip. Band-level groupings in the summary do not constitute individual identification.

Use Case

Teacher awareness — for example: "3 students have audio off — make sure instructions are also written on the board." This information is for pedagogical support only. It is not used for assessment, grading, or any evaluative purpose.
