Application Rules
🏗️
Applied to route wrapper, not body
Background classes are applied to <main> or the route-level wrapper element — never directly on <body>. This allows the body to remain neutral for modals and overlays that should escape the route context.
<main class="bg-teacher-home"> ... </main>
📐
Always min-height: 100vh
All background wrappers set min-height: 100vh to prevent gradient clipping on short-content pages. Combined with position: relative to anchor particle layers.
✨
Particle effects: mount / unmount lifecycle
Star and particle effects are generated by JavaScript on component mount and removed on unmount. Use requestAnimationFrame for the animation loop; cancel it with a ref on unmount. Checking prefers-reduced-motion before starting animations is required.
if (window.matchMedia('(prefers-reduced-motion: reduce)').matches) return;
♿
prefers-reduced-motion: disable all animations
When reduced motion is preferred, particle animations are skipped entirely (particles still render statically if desired, but no movement). CSS keyframe animations on .star-particle are also suppressed via the media query override.
🖨️
Print media: backgrounds stripped to white
All [class*="bg-"] elements are overridden to background: white !important in the print stylesheet. Particle canvases should also be hidden with display: none during print.
🔲
Forced Colors (High Contrast): black background
Windows High Contrast and other forced-color environments receive background: #000000 !important. Text and interactive elements rely on the OS-defined system colors for sufficient contrast.
🏷️
Band class determined at play session start
The correct .bg-play-p* class is resolved from the child's enrolled band (P0–P3) before the play route mounts. The class is passed as a prop to the route wrapper; it does not change during a session.
const bgClass = `bg-play-p${child.band}`;
🎨
Persona tint via pseudo-element, not additional class
The subtle radial tint on .bg-teacher-home, .bg-parent-home, and .bg-owner-home is applied via a ::before pseudo-element with a radial gradient overlay. This avoids stacking additional DOM nodes.
Particle System — JS Reference
/** * generateParticles(canvas, color, count, options) * * @param {HTMLCanvasElement} canvas — target canvas (position:absolute, inset:0) * @param {string} color — hex color for particles * @param {number} count — number of particles (default: 60) * @param {object} options — { minSize, maxSize, speed, opacity } * * Returns a cancel function: () => void * Call it on component unmount to stop the rAF loop. */ function generateParticles(canvas, color, count = 60, opts = {}) { // Bail immediately if reduced-motion is preferred if (window.matchMedia('(prefers-reduced-motion: reduce)').matches) return () => {}; const ctx = canvas.getContext('2d'); canvas.width = canvas.offsetWidth; canvas.height = canvas.offsetHeight; // Build particle array const particles = Array.from({ length: count }, () => ({ x: Math.random() * canvas.width, y: Math.random() * canvas.height, r: 0.5 + Math.random() * 1.5, dy: -(0.1 + Math.random() * 0.2), alpha: 0.1 + Math.random() * 0.5, phase: Math.random() * Math.PI * 2, })); let raf; function draw() { ctx.clearRect(0, 0, canvas.width, canvas.height); particles.forEach(p => { p.y += p.dy; p.phase += 0.02; if (p.y < -4) p.y = canvas.height + 4; ctx.beginPath(); ctx.arc(p.x, p.y, p.r, 0, Math.PI * 2); ctx.fillStyle = color; ctx.globalAlpha = p.alpha * (0.6 + Math.sin(p.phase) * 0.4); ctx.fill(); }); ctx.globalAlpha = 1; raf = requestAnimationFrame(draw); } draw(); return () => cancelAnimationFrame(raf); } // Band colour map const BAND_PARTICLE_COLOR = { p0: '#ffd166', // Pre-K gold p1: '#9b72ff', // K–1 violet p2: '#58e8c1', // G2–3 mint p3: '#ff7b6b', // G4–5 coral };
