Notes

Audio context Created on first user gesture to comply with autoplay policies. Shared across checks.
LUFS estimation Approximated from AnalyserNode peak amplitude; not a true LUFS meter (no K-weighting). Sufficient for go/no-go level check.
Mic permission getUserMedia is optional. If denied, only the voice-answer feature is affected. Core play is unaffected.
Latency baseline 300ms target matches Chrome's typical AudioContext startup. High latency may cause perceptible delay on first question playback.
No timer This tool has no countdown or auto-expiry. Beta testers can re-run at any point.
