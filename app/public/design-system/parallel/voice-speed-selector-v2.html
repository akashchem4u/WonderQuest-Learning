Speed Values
Normal speed SpeechSynthesisUtterance.rate = 0.9
Slow speed SpeechSynthesisUtterance.rate = 0.75
Pre-recorded assets Available at Normal speed only; Slow = always TTS fallback
Lock Behaviour
Locked state voice_speed_locked = true → child speed chip hidden; always plays at voice_speed value
Unlocked state Speed chip shown in top-right of play screen; tap → 2-option picker; selection stored to child_audio_prefs immediately
Database Schema
child_audio_prefs.voice_speed ENUM('normal', 'slow') child_audio_prefs.voice_speed_locked BOOLEAN
Default by Band — assignBandDefaults(band_id)
function assignBandDefaults(band_id) { // P0 — Pre-K Starters if (band_id === 'P0') return { voice_speed: 'slow', voice_speed_locked: true }; // P1 — Adventurers if (band_id === 'P1') return { voice_speed: 'slow', voice_speed_locked: false }; // P2 — Explorers if (band_id === 'P2') return { voice_speed: 'normal', voice_speed_locked: false }; // P3 — Trailblazers if (band_id === 'P3') return { voice_speed: 'normal', voice_speed_locked: false }; // Fallback return { voice_speed: 'normal', voice_speed_locked: false }; }
In-Play Toggle Flow
Condition If voice_speed_locked === false → speed chip rendered in play screen top-right
Interaction Tap chip → 2-option picker (Normal / Slow); selection persisted to child_audio_prefs immediately; audio restarts at new speed
Locked Chip hidden; playback always uses voice_speed parent value; child has no control
