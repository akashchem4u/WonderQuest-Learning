Mobile vs Desktop Differences

  • Mobile: single column; Desktop: 2-col grid
  • Mobile headline: 1.8rem; Desktop: 2.4rem
  • Mobile stats: horizontal-scroll chips; Desktop: vertical list
  • Mobile CTAs: full-width stacked; Desktop: side-by-side row
  • Mobile segs: 7px; Desktop: 8px
  • Mobile phone frame: 390×700px; Desktop: max-width 960px

Stat Chips

  • Horizontal scroll row: overflow-x: auto; scrollbar-width: none
  • Each chip: flex-shrink: 0; min-width: 80px
  • Val: 1.1rem Nunito 900; Label: 0.65rem
  • Perfect bonus chip: gold bg tint + gold val
  • ❌ Never show wrong-answer chip
  • ❌ Never show accuracy % chip

Touch Targets

  • Primary CTA: 54px height (safe for thumb tap)
  • Secondary CTA: 46px (smaller but still ≥44px)
  • -webkit-tap-highlight-color: transparent on both
  • Gap between primary and secondary: 8px
  • CTAs always at bottom of phone inner — margin-top auto

Star-Safe

  • Same star-safe rules as desktop version
  • "Play Again" → same star-safe overlay (see desktop spec)
  • Star count chip always shows positive delta (+N ⭐)
  • ❌ Never negative delta
  • ❌ Never accuracy %
