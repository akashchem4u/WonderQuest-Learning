Support Tips

Tips are generated per skill-category from a curated tip library (not AI-generated per child). Each tip is a specific, actionable at-home activity. Tips must be:

1. Framed as something FUN to do together (not "drill your child on X").

2. Context-appropriate (pre-K tips differ from Grade 4–5 tips).

3. Free / no materials required unless clearly stated.

Empty State

If 0 skills qualify: show "🎉 All skills are looking strong this week!" — never show an empty card with no message. This positive empty state is a reward for the parent too.
