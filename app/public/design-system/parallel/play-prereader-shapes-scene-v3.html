📋 play-prereader-shapes-scene-v3 · Engineering Spec

Purpose

  • Pre-K shape recognition play scene — child sees a shape hero and must find matching shape in 2×2 grid
  • Gold (#ffd166) Pre-K theme throughout — topbar gradient, progress bars, hero border, CTA button
  • No timer, no accuracy %, stars never decrease

Tabs / States

  • Active Shape hero visible with gold glow animation, 2×2 shape choice grid, bottom 3 action buttons
  • Correct Hero border → green, correct btn gets selected-correct, others get dimmed, gold CTA appears
  • Wrong Wrong choice gets selected-wrong + shake, red banner with actionable hint ("Run your finger around it!"), star-safe shown
  • Spec This document

Shape Hero

  • 160×160px card, border-radius: 20px, 3px gold border
  • SVG shape centered, ~100px fill area
  • letter-glow animation: box-shadow pulse 2s ease-in-out infinite
  • On correct: border-color → #50e890, background → #0d2a1a, correct-glow keyframe fires once

Shape Choices

  • 2×2 grid, each button 90px tall, 3px border, border-radius: 16px
  • SVG shape 42px + shape label text below
  • obj-pop entry animation staggered 0.05s per button
  • Shapes: Circle / Triangle / Square / Star (one correct, three distractors)
  • Wrong state: selected gets shake + red border, others remain active (retry freely)
  • Correct state: correct gets green border, others get dimmed (opacity 0.35, pointer-events none)

Progress Strip

  • Segment bars (not dots) — flex:1 per question, 6px height
  • Gold theme: done=#ffd166, active=rgba(255,209,102,0.7), pending=rgba(255,209,102,0.2)

Bottom Bar

  • Active/wrong: 3 flex:1 buttons — 🔊 Replay / 💡 Hint / 🤷 IDK yet
  • Star-safe strip below buttons on active and wrong states
  • Correct: single gold CTA button "Next Shape ✨ +1⭐"

Language Rules

  • Wrong feedback: "Not quite — look at the shape up top." + actionable physical prompt
  • Never "Wrong!" or "Incorrect!"
  • Coach speaks in repetition: "Circle! Circle!" — reinforcement not testing
  • Correct: "Yes! Circle! Round all the way around…"

Design Tokens (Pre-K Gold Theme)

  • Topbar gradient: linear-gradient(135deg, #2a1800, #1a0c00)
  • Gold accent: #ffd166 — progress segs, hero border, CTA button
  • CTA button: linear-gradient(135deg, #ffd166, #f0a000), color #1a0c00
  • Correct: #50e890 border / #0d2a1a bg
  • Wrong: #ff7b6b border / #2a0e0e bg
  • Star-safe badge: #1a2a15 bg / #50e890 border
