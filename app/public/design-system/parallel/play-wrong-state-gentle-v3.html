📋 play-wrong-state-gentle-v3 · Engineering Spec

Purpose

  • Standardized gentle wrong state — applied across all play scenes via CSS + JS class swap
  • Critical principle: wrong state NEVER shames. Every wrong state gives a specific, actionable hint.
  • Stars never decrease. Star-safe badge always visible on wrong state.
  • Other answer cards remain fully active for retry — child chooses again immediately

Forbidden Language (NEVER use)

  • ❌ "Wrong!" / "Incorrect!" / "Oops!"
  • ❌ "You missed it" / "You failed" / "Try harder"
  • ❌ "That's wrong — the answer was X" (never reveal answer preemptively)
  • ❌ Strike count display ("2 strikes left")
  • ❌ Red X animation on child's face/avatar

Approved Wrong Language Patterns

PatternExample
"Not quite — [actionable hint]""Not quite — say 'bee' out loud"
"That wasn't it — [specific cue]""That wasn't it — look at the highlighted sentence"
"Let's [do strategy] — [encouragement]""Let's count again — touch each star one at a time!"
"Give it another try!"Used alone or after hint sentence

Wrong Banner Structure

  • Border: 2px solid #ff7b6b — consistent across all bands (wrong color doesn't change)
  • Background: #2a0e0e
  • Icon: question-type appropriate (⭐ for counting, 🔊 for phonics, 📖 for reading, 🔍 for inference)
  • Headline: 0.92rem Nunito 900, coral color — the hook
  • Hint: 0.84rem Nunito 700, white — specific actionable instruction

Wrong Card CSS

  • Class .wrong-card on the selected wrong answer card
  • Background: #2a0e0e, border: #ff7b6b, text: #ff7b6b
  • ac-shake keyframe fires once (translateX ±8px, 0.5s ease-out)
  • Other cards remain default style — fully interactive for retry

Coach Role in Wrong State

  • Small inline coach (28px avatar, 0.82rem text) in supporting role — not dominant
  • Coach reinforces the STRATEGY (count with finger, listen for the sound) not just "try again"
  • Coach does NOT repeat the wrong banner text — adds a complementary angle

Bottom Bar on Wrong State

  • Buttons remain: 🔊 Replay / 💡 Hint / 🤷 IDK yet — all 3 available
  • G2-3 and G4-5 reading: 🔊 Replay → 📖 Re-read (text-primary scenes)
  • Star-safe badge below buttons — explicit reassurance
