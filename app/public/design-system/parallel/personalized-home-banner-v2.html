Data model

personalized_home_banners
Stores variant, child_id, reason_text, payload JSON, and expiration window.
banner_impressions
Logs impression time, route, and whether the banner was dismissed or clicked.
banner_suppressions
Tracks child, variant, and suppression window after repeated dismissals.

Variant rules

Interest Match uses stated interests first. Resume Path uses the last successful checkpoint. Celebrate can trigger only after a genuinely positive milestone and may not mention leaderboards or peer comparison.

Copy rules

Child route: first name only, positive tone, no scarcity. Parent route: plain-language reason, controls, and visible suppression options.

Routing note

Banner selection must be deterministic for the current home load so the child does not see the banner change while scrolling or returning from a modal.
