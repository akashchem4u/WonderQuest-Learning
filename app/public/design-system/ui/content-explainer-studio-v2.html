⚙️ Engineering — Spec Reference This tab is the canonical spec for the Explainer Studio. All API routes, script token formats, QA rules, and design tokens are defined here. Content-admins cannot publish directly — all publish actions route through owner approval. Hard QA blocks gate the "Send to Owner Review" button client-side and server-side.
API Routes
GET GET /content/explainer/:id — fetch full explainer node (script + hints + QA state)
PUT PUT /content/explainer/:id/script — save script; returns rule violations synchronously
POST POST /content/explainer/:id/hints — add or update a media hint
POST POST /content/explainer/:id/qa-submit — run full QA check; returns qaReport
Script Token Format
• [WORD: "x"] — renders as a large highlighted word card in child view
• [PAUSE] — inserts a voice-coach beat pause (500ms default)
• Emoji inline — rendered as-is; emoji-first rule requires opening line to begin with emoji
• Plain text — rendered with Sora font, sized by band context (P0 larger)
Hard QA Blocks (Publish Gated)
❌ Missing correct answer on any question
❌ Audio hint without written transcript
❌ Duplicate answer options in any question
❌ Prohibited or copyrighted content detected
❌ Shame language or grade labels in script or question text
Warnings (Non-Blocking)
⚠️ Borderline word length — sentence near but under limit
⚠️ Image hint missing alt text (accessibility advisory)
⚠️ Flagged hint card — requires manual review before next publish
⚠️ Script character count above 350 (approaching 400 max)
Publish Gate Flow & Word Limit Rules
1. Content-admin creates / edits script and hints → saves draft (auto-check runs)
2. Content-admin submits to QA via qa-submit endpoint
3. If hard blocks exist: blocked. Content-admin must resolve and re-submit.
4. If all blocks cleared: "Send to Owner Review" becomes enabled
5. Owner approves or requests changes — content-admin cannot self-publish
P0 Pre-reader band: max 8 words/sentence; auto-check on every save
P1 K-1 band: max 15 words/sentence; auto-check on every save
• Media hints: max 2 per question; hints must not directly reveal answer
• Script character limit: 400 characters total
