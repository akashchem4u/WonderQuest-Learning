child-badge-detail-overlay-v2 — Specification

Persona: Child · Mobile 375px · Nunito 900 · #100b2e bg · overlay card style

API: GET /child/badge/{id} → { name, emoji, description, story, earned_at, state, unlock_steps[] }

API: POST /child/badge/{id}/share { channel }

  • Earned badge: large glowing badge circle, name, description, how-earned story, date, share CTA
  • Locked badge: emoji steps to unlock, replay motivation, NO shame language ("can't yet" is banned)
  • Locked badge framing: "Here is how to unlock it" not "You haven't done this yet"
  • Share panel: optional — always surfaced as opt-in, never required
  • New badge celebration: confetti emoji, glow animation, XP chip, collect CTA
  • Overlay style: dim background, bottom-sheet rounded card, drag handle
  • No timers, no accuracy %, no grade labels, replay can never lose stars
