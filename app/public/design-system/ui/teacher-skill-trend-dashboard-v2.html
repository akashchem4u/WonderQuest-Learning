Component Spec: Skill Trend Dashboard v2
  • Desktop 900px+, Sora, bg #f0f4f8, nav #2a5080
  • Tab 1: headline stats + SVG polyline trend per skill — up/flat/down direction indicator
  • Tab 2: heat map grid, skills as rows, weeks (W1–W8) as columns; cell = class avg score
  • Heat color scale: red (<40%) → amber (50–60%) → light green (70–80%) → dark green (90%+)
  • Tab 3: band comparison — 2×2 grid, P0/P1/P2/P3, each shows Flag/Watch/Strong distribution bars
  • Tab 4: auto-generated insights with Alert (red), Positive (green), Observation (blue) tags
  • Filter chips: All / P0 / P1 / P2 / P3 — each color-coded to match band
  • SNS tier colors: Flag #9a2a2a, Watch #c07800, Strong #1a8050
  • SVG trend lines are inline; no external chart library required
  • Export button in header for CSV download of heatmap data
