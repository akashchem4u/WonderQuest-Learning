Engineering Spec — Teacher Assignment Creator v2 Full API surface, data model, business rules, and design tokens for the assignment creation workflow. This tab is for internal engineering reference only — not shown in production UI.
API Routes
  • GET /teacher/skills — List available skills with band, description, question_count
  • POST /assignments/create — Create new assignment (draft | scheduled | active)
  • GET /assignments/:id — Fetch single assignment with students and status
  • PUT /assignments/:id — Update assignment settings or reschedule
  • DELETE /assignments/:id/draft — Discard a draft assignment permanently

Auto-Hint Override Rule

Students with sns_tier: flag (Needs Support) always receive +1 additional hint on every question, regardless of the assignment's difficulty setting. This override is applied server-side at question delivery time and is not configurable by the teacher.
