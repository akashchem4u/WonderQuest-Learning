Component Spec: Assignment Review Desktop v2
  • Desktop 900px+, Sora, bg #f0f4f8, nav #2a5080
  • Two-panel layout: 300px scrollable list left, fluid detail panel right
  • List filter chips: All / Sent / Queued / Draft / Overdue — each shows count
  • List items: name, status chip, skill/band, learner count, mini progress bar, due date
  • Overdue items: red left border + light red bg; selected items: blue left border + blue tint
  • Detail panel: assignment info card, learner group card with avatar row, completion preview bar, schedule
  • Completion bar: segmented green/amber/red showing done/in-progress/not-started
  • Tab 3: full per-learner completion list with status chip
  • Tab 4: edit form — name, band, group, new due date; Delete / Save & Resend actions
  • Contact-safe: learner first name + last initial only; no parent contact details shown
