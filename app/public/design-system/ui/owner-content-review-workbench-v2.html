Tab 5 — Spec Notes
Route & Layout
Route: /owner/content/review. Desktop-first 3-col layout. Mobile collapses to queue list → detail push screen. Left col: 280px, right col: 340px, center: flexible. Min desktop width: 960px.
Review Queue Sort Order
Default: Flagged items first (sorted by report count desc), then New content (sorted by submission date desc), then Revision returns. Filter chips override this. QA fail always surfaces a content item to top of its section.
