Before You Start

These scenarios are designed for classroom or small-group use. Run them during a normal practice session where possible — avoid staging artificial conditions.

1
First Class Setup
Onboarding · 15–20 min
Set up WonderQuest for a new class group. Add students, set grade level, assign initial content band.
👁️ Watch for: how many steps to add 5 students? Any confusing field labels? Confirmation feedback clear?
2
Support Queue
Intervention · 10 min
Navigate to the support queue and identify the top 3 students needing intervention this week.
👁️ Watch for: is the queue sorted helpfully? Can you drill into a student in under 2 taps?
3
Intervention Plan
Intervention · 10 min
Create an intervention plan for a struggling student. Set goal, content, and frequency. Save and verify it appears in the student's card.
👁️ Watch for: does the plan editor make sense? Is saving/confirmation clear?
4
Parent Message
Communication · 5 min
Send a message to a parent about their child's recent progress. Use the message center to compose and send.
👁️ Watch for: is the student lookup fast? Does the message thread feel appropriate for parent communication?
Scenario 1 of 4
First Class Setup
Onboarding · Est. 15–20 min · Class of 5+ students
Steps
  1. 1Open teacher dashboard. Navigate to Class Setup from the home screen.
  2. 2Create a new class. Enter class name and grade level.
  3. 3Add at least 5 students using the student import or manual entry flow.
  4. 4Assign an initial content band (Reading or Math) to the class.
  5. 5Verify the class appears on your dashboard with correct student count.
Observation Prompts
How many total taps to add 5 students? · Were any field labels confusing? · Did the confirmation feel trustworthy? · What would you change first?
Note

If you already have a class set up, try adding a new one with a fresh group. The first-run experience is what we're testing.
