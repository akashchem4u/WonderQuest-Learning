Here are some ideas to support Emma at home this week 🏠
These activities are quick, low-pressure, and based on Emma's recent sessions in Ocean World. Even 10 minutes a day makes a real difference — and they're designed to be fun for both of you!
🎵
Sound Stretching Game
⏱ ~10 min
  1. Pick 5 short words together (e.g. "ship", "rock", "fish").
  2. Say a word slowly and stretch out each sound together — "sss-hh-ip!"
  3. Take turns guessing what the stretched word is.
  4. Celebrate each one with a high-five! 🙌
Supports: Phoneme Blending
Why this helps: Emma is building confidence connecting sounds in longer words — hearing them stretched out helps that click happen naturally.
📖
Ocean Word Hunt
⏱ ~10 min
  1. Find a picture book or magazine page together.
  2. Hunt for words that have a vowel team (like "ee", "ai", "oa").
  3. Circle them or point them out — no writing needed!
  4. Bonus: try reading just those words out loud.
Supports: Vowel Patterns
Why this helps: Emma is building familiarity with vowel combinations. Spotting them in real books makes the pattern feel natural and rewarding.
🃏
Sight Word Flashcard Dash
⏱ ~8 min
  1. Write 8–10 sight words on index cards (e.g. "the", "said", "they").
  2. Hold up one card at a time — Emma reads it fast.
  3. Make it silly: she earns a point for each one, you guess one too!
  4. Track her record each week to see progress.
Supports: Sight Words
Why this helps: Recognizing these common words automatically frees up Emma's brain to focus on understanding stories — a big deal for fluency!
🌊
Read-Aloud Story Time
⏱ ~15 min
  1. Choose any book Emma enjoys — ocean theme even better!
  2. Take turns: you read a page, she reads a page.
  3. After each page, ask: "What happened? What do you think comes next?"
  4. End with "What was your favorite part?"
Supports: Reading Fluency · Story Comprehension
Why this helps: Reading together builds fluency, comprehension, and most importantly — a love of stories. This one goes a long way!
✨
These ideas are based on Emma's recent sessions and current skill focus areas. They'll refresh each week as Emma grows! No pressure — even one activity makes a difference.
🐙
Teacher Tip from Ms. Rivera
After reading together, ask Emma about her favorite ocean creature and have her describe it in a few sentences. It's a wonderful way to build vocabulary and her confidence in expressing ideas out loud!
Tab 5 — Engineering Spec
API routes, layout breakpoints, parent language rules, tier label reference, session data rules, and practice idea generation notes.
WonderQuest
Spec
🧑

API Routes

  • GET /parent/child/:id/overview — child card, quick stats, hero summary, wins, support areas, sparkline data, next-focus recommendation
  • GET /parent/child/:id/skills — full skill list with tier labels, forward-looking notes, 4-week trend data (activity booleans only)
  • GET /parent/child/:id/sessions — paginated session list; fields: world name, date, highlight strings, stars earned (int), duration bucket, skills practiced list, badge earned (nullable)
  • GET /parent/child/:id/practice — weekly practice ideas; auto-generated from SNS tier data; teacher overrides included if present

Desktop Layout

  • Outer shell: max-width: 1160px centered desktop frame
  • 2-col CSS grid: grid-template-columns: 280px 1fr
  • Left sidebar: child card + quick stat chips + recent badge
  • Responsive breakpoint at 768px: sidebar collapses to top strip (grid becomes single col), stat chips flex-wrap to row
  • Secondary breakpoint at 900px: wins grid and practice grid collapse from 2-col to 1-col

Parent Language Rules (STRICT — enforced across all tabs)

  • NEVER show accuracy %, raw scores, or numeric performance figures
  • NEVER use the words "missed", "behind", "failed", "wrong", or "weak"
  • NEVER use grade labels (e.g. "Kindergarten level", "Grade 2") — use world names only
  • ALWAYS use forward-looking, growth-oriented language ("building", "exploring", "next focus", "great progress")
  • ALWAYS frame support areas as opportunities, not deficits
  • Session duration shown as approximate bucket only ("~15 min") — never exact seconds
  • Stars earned per session are fine to show; no other numeric performance metric is permitted

Tier Label Reference

Tier Label (Parent-Facing) Text Color Background Parent Description
support Support Area #c07800 #fff4d0 Emma is building this skill — great focus area for home practice
growing Growing Strength #1a7060 #d0f0ea Emma is making steady progress here — keep encouraging her!
strength Strength #1a5030 #d0f0e0 Emma has really got this one! 🌟

Session Data Rules

  • Stars earned per session: show as integer (e.g. "+6 stars earned") ✅
  • Duration: show as approximate bucket only — "~15 min", "~10 min" etc. ✅
  • Accuracy %: NEVER shown ❌
  • Raw score or correct/incorrect counts: NEVER shown ❌
  • Session highlights: human-readable strings generated server-side in parent language
  • Expanded detail: skill name + forward-looking note + badge earned (nullable) + XP earned (OK to show XP)

World Names (ONLY valid world references)

  • Dream World · Ocean World · Fire World · Star World · Crystal World
  • Forbidden: any grade label, reading level label, or age bracket label as a world identifier

Practice Ideas — Generation Rules

  • Auto-generated weekly from the child's current SNS tier data — prioritizes support tier skills
  • Each practice card maps to exactly one (or two) skills via practice_skill_tags[]
  • Teacher can override or append practice cards via the teacher dashboard — teacher-authored cards are flagged with source: "teacher" and shown with the teacher tip styling
  • Practice cards refresh every Monday at 00:00 UTC
  • Steps are plain language, no edu-jargon — max 4 steps per card
  • Time estimate is a bucket string: "~8 min" / "~10 min" / "~15 min"

Child Selector Dropdown (Nav)

  • Populated from GET /parent/children — returns all linked children for the authenticated parent
  • Switching child re-fetches all tab data for the new child_id
  • Default child: first in list, or last viewed (stored in localStorage)
