Design spec for play-question-card-grid-v3.

Component Identity

Fileplay-question-card-grid-v3.html
Versionv3 (refined from v2)
Width375px mobile
Grid2x2, equal columns

Card States

Default#1a1240 bg, #2a1f5e border
Hover/tap#221a50 bg, violet border
Selected#2a1f00 bg, gold border + glow
Correct#0a2e25 bg, mint border + pop anim
Wrong#2e0a0a bg, coral border + shake anim
Disabledopacity 35%, pointer-events none

Rules

Band accent (4px bottom border) matches skill band color on correct card only. Wrong answer: shake animation 400ms, then disabled state. No "X" or "wrong" text beyond icon. No star penalty visible on wrong answer. Cards lock (disabled) after any answer submitted — no re-selection. Correct card always revealed after submission regardless of selection.
