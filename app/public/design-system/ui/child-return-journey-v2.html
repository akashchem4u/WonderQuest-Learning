Panel 5 — Spec Notes: Canonical spec for child return journey. Route: /play (return state, detected on session start if last_session > threshold). Child theme throughout.

Child Return Journey — Spec

Return state triggers

Days since last sessionStateTonePrimary CTA
0–1 dayNormal session start (no return screen)—Normal play entry
2–3 daysShort awayWarm welcome + streak check"Jump back in!"
4–6 daysMedium awayGentle return + streak reframe if broken"Continue / Fresh start"
7–13 daysLong awayBig return celebration + new content"Start today's quest"
14+ daysEpic returnMax celebration, no shame, full reorientation"Let's do this!"

Streak rules

ConditionDisplay
Streak intact (returned within grace period)🔥 "Streak still going!" — show current count
Streak broken (returned after grace period)🌱 "Fresh start!" — show day-1 progress bar toward next milestone
No streak yet (new user)🌱 "Start your first streak!" — same new-streak UI

Resume vs. fresh start

  • 2–3 day away: show resume only (single CTA)
  • 4–6 day away: show resume primary + "start fresh" secondary link
  • 7+ day away: show "start today's quest" primary + resume as secondary option
  • Never force the child to decide between options upfront — lead with the recommended action

"While you were away" items

  • Show max 2–3 items — unclaimed trophies first, new content second
  • Only show if away 3+ days
  • Never block resume CTA — discoveries appear below the primary action
  • Unclaimed trophies: tap → opens trophy claim animation inline
