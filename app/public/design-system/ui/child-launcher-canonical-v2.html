Tab 5 — Spec
Route & State Routing
Route: /child or /play entry. Launcher shown when: no active session in last 5 min OR child explicitly exits to home. State determined server-side on load.
Launcher State Decision Tree
ConditionState shown
child.sessions === 0First Time — avatar + name entry
last_session < 1 dayResume — current node highlighted
last_session 1–3 days AND streak intactComeback short-away — streak card
last_session 4–13 daysComeback mid-away — resume + new content
last_session 14+ daysComeback long-away — fresh start + new content
device has multiple child profilesMulti-child tile picker (overrides all above)
Reading Load Rules
Ultra-low reading load: maximum 8 words on any CTA. Primary CTA always the largest element on screen. Emoji carries the primary message. Never show skill names in launcher — world names and emoji only. Numbers (streak, XP) shown numerically, not spelled out.
