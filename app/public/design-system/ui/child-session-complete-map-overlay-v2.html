Map layout on overlay

  • Mid-skill: vertical node trail (up to 5 nodes tall, scroll if needed)
  • Skill mastered: horizontal mini-map (all nodes in a row) — more compact, less emphasis on trail
  • Band complete: mini skill list (names + ✓ + badge icon) — no node trail needed

Never show on overlay

  • Accuracy % (correct out of total)
  • "X wrong answers" or any error count
  • Time taken
  • Comparison to other sessions ("better than last time")
