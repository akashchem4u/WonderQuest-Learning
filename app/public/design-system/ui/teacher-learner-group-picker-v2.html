Component Spec: Learner Group Picker v2
  • 390px mobile, Sora, bg #f0f4f8, nav #2a5080
  • Three card options: Whole Class, Support Tier (Flag+Watch), Custom
  • Each card shows learner count and avatar row (initials only, contact-safe)
  • Tier counts shown with SNS color coding: Flag #9a2a2a, Watch #c07800, Strong #1a8050
  • Selected card gets highlighted border and blue background tint
  • Tab 2: scrollable whole class list with tier badges
  • Tab 3: support tier split into Flag and Watch sub-groups
  • Tab 4: checkbox multi-select with search filter and live count
  • Bottom bar: Cancel + Confirm — Confirm label reflects selection
  • Contact-safe: first name + last initial only, no contact details shown
