API: PATCH /owner/feedback/{id} { state, assignee } · POST /owner/feedback/{id}/link-pr { pr_url }

  • P0 chip: red bg #4a0a0a / text #ff6b6b · P1: amber · P2: teal · P3: blue
  • Severity filter chips: toggle-able, active state fills chip with severity color
  • Bulk action bar: select-all, bulk-assign, bulk-resolve, export
  • SLA rules: P0=4h · P1=24h · P2=72h · P3=7d — show overdue/at-risk/on-track in SLA Tracker tab
  • Assignee board: 4-col team grid with queue count chips per severity
  • Severity mix tab: SVG bar chart by P0/P1/P2/P3 counts
  • Actions per row: Assign / Resolve / Link PR
