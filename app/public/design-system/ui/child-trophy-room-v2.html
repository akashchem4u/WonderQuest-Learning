child-trophy-room-v2 — Specification

Persona: Child · Mobile 375px · Nunito 900 · #100b2e bg · #ffd166 gold

API: GET /child/trophies returns { world_trophies[], reading_trophies[], math_trophies[], streak_trophies[] }

  • Trophy groups: World / Reading / Math / Streak — each as a card with progress bar and mini badge row
  • Locked trophies: grayscale silhouette emoji + friendly "keep playing" prompt
  • NEVER shame language: no "you failed to" or "missed" — always "keep going" framing
  • Streak trophies: show current streak count prominently at bottom of streak tab
  • Progress bars: gold for world, violet for reading, mint for math, coral for streak
  • Locked trophy question mark replaces badge icon — motivating, not punishing
  • No timers, no accuracy %, no grade labels anywhere
