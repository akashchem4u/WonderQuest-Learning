📡 Live Status Strip — Spec Always-visible bar at the bottom of the hero section (above launcher cards). Pulls from real-time platform stats. If a platform incident is active, the first cell changes from green dot to amber/red + incident label.
Live Status Strip
Bottom edge of hero, above launcher cards. Horizontal scroll on mobile.

Cells

System Status
Green dot + "All systems operational" — changes to amber (degraded) or red (outage) when incident active. Source: status API endpoint.
Quests completed today
Live count, resets at midnight UTC. Shows rolling 24h if preferred. Drives social proof.
Active learners right now
Current concurrent active session count. Fuzzy rounded to nearest 10 for privacy.
Badges earned this hour
Rolling 60-minute badge grant count. Adds energy to home page visit.
Compliance signal
Static: "COPPA · FERPA compliant". Not dynamic. Always shown.

🔒 Trust & Safety Strip — Spec Footer of the home page. Six fixed items. Static — no API. Calms parental anxiety before sign-up. Keep copy short (3–4 words each). No icons that look like badges/certifications (avoid false impressions).
Trust & Safety Strip
Footer of home route. Static. Six items max.

IconCopyHover tooltip
