Description
Audio response delay exceeding 800ms affecting ~34% of play sessions on iOS 17+. Root cause traced to AVAudioSession initialization order in voice coach bridge. Fix involves lazy-loading audio context after session warm-up. Regression test coverage needed for Android WebView path.
