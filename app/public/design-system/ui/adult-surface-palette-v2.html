Desktop vs Mobile Surface Notes
Home: Desktop uses centered max-width 1080px container, 3-col feature grid on surface-2 background. Mobile uses full-bleed card stacking.

Parent: Desktop: left sidebar (surface-2) + main panel (base-bg) + optional right activity panel. Mobile: single column, all cards full-width. Tier badges remain same colors on both.

Teacher: Desktop: fixed left nav (surface-2, 220px) + main content area (base-bg) + optional detail panel. Student list uses compact table rows on desktop, card rows on mobile. Intervention timeline always full-width.

Owner: Desktop: top-level nav bar (surface-2) + content area (base-bg). Route health uses multi-column grid. Feedback queue and detail use a master-detail split on desktop (queue list left, detail right). Mobile collapses to stack.
