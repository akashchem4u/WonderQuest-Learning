Coverage Thresholds
Target: 8 questions per node — ideal for adaptive variety.
OK minimum: 6 questions — sufficient for most sessions.
Warning (<6): Session may begin to repeat questions for active students. Add content soon.
Critical (<4): Node will run dry mid-session for students actively using it. Blocks release if any active node is critical.
Empty (0): Node not yet published. Not a blocker if node is not in active use.
