⚙ TAB 5 — Engineering Spec: Child Streak Comeback / Return Flow
API Routes
GET /player/return-state → returns gap_days, streak, last_session, comeback_reward if applicable
POST /player/session-start → triggers gap detection; response includes return_flow flag + flow_type
GET /player/streak → current streak count, last_played_at, streak_safe (bool)
Gap Detection Logic
Gap (days since last session)Flow shownStreak actionReward granted
0 – 1 daysNone — normal play, no return flowStreak continues as-isNone
2 – 6 daysTab 1 — Short Gap return flowStreak preserved (STREAK SAFE)None
7 – 29 daysTab 2 — Week Gap return flowStreak resets to Day 1+1 ⭐ Comeback Star · +30 XP
30+ daysTab 3 — Long Gap return flowStreak resets to 0, shown as fresh start+2 ⭐ Explorer Stars · +100 XP · 🏅 Return Badge
Streak-Safe Language Rules
NEVER use "lost your streak", "broken streak", "streak at risk", "you missed", "you failed"
NEVER show countdown timers or urgency pressure ("only X hours left")
Always use: "fresh start", "new streak begins", "comeback", "starting fresh", "adventure continues"
Streak reset is framed as a celebration event — reward, not punishment
Flame emoji 🔥 used for active streak; star ⭐ used for neutral restart (30d+ gap) — avoids punitive flame-extinguish imagery
All gap handling flows end with a forward-looking CTA — never backward-looking recap of absence

Child Safety Rules
Zero shame framing — child is never made to feel bad for time away
All return flows are celebratory: returning is rewarded, not just normalized
No dark patterns: no countdown timers, no guilt messaging, no streak insurance mechanics targeting anxiety
Calm, warm tone throughout — ages 5–10 audience, Nunito 900, emoji-forward
