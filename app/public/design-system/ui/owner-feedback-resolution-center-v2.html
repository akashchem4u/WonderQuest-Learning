Bulk Actions
Activate via "Select" button in topbar. Multi-select checkboxes appear on rows. Bulk actions available: Archive selected (resolved items only) · Assign all selected · Change severity (all selected). Bulk resolve not available — each item must be resolved individually with a note.
Resolution Note Rules
Resolution notes are required when: (1) marking any P0 or P1 as resolved; (2) downgrading severity. Optional for P2. Note persists in audit trail and is visible to team. Max 500 characters. Verification method radio is required for P0/P1 resolve.
