Component Spec: Student History Desktop v2
  • Desktop 1024px+, 3-column grid layout (240px | 1fr | 300px), Sora, bg #f0f4f8
  • Left col: profile card (avatar, name, tier badge, stats), scrollable session list
  • Center col: 8-week SVG bar chart, tier history timeline with colored dots, skill breakdown bars
  • Right col: append-only intervention log timeline, teacher notes cards
  • All 3 columns scroll independently within the fixed viewport height
  • Tab 2: full session table — date, skill, band, duration, score, hints used, tier at time
  • Tab 3: expanded skill progression — 2-up grid of bar charts per skill
  • Tab 4: side-by-side full intervention log + all notes
  • Tier dots: Flag #9a2a2a, Watch #c07800, Strong #1a8050
  • Contact-safe: parent contact shown as WQ-routed only; no phone/email
