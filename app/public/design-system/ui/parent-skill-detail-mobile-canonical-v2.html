Tier language rules

TierBadge textPlain descriptionProgress barPrimary CTA
Support "Building this skill" Short, normalizing — "still developing, practice helps" Coral fill, ~28% View teacher's plan
Strength "This is a strength" Celebratory — "doing well, strong foundation" Mint fill, ~82% Message teacher
Next "Almost there!" Encouraging — "making great progress, close to next level" Blue fill, ~55% See what's coming next
Language rules: NEVER say "X% correct," "missed 3 days," "accuracy dropped," or "failed." Always use growth framing. Support = normalizing + actionable. Strength = celebratory. Next = encouraging + forward-looking.
