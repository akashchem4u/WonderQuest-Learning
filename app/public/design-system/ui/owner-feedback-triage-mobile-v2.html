Spec: Feedback Triage Mobile v2
  • 390px mobile, Sora, bg #0d1117, gold #ffd166
  • Items sorted P0 → P1 → P2; within tier sorted by date (newest first)
  • Each item: ID, severity chip, state chip, 1-line summary, meta (date/assignee), action buttons
  • Severity chips: P0 red, P1 amber, P2 blue
  • State chips: Open (grey), Assigned (green), In Review (bright green), Blocked (red)
  • Action buttons per state: Assign / Escalate (unassigned), Resolve / View (assigned/review)
  • Swipe gesture visual hint: → resolve, ← escalate — shown as buttons in non-gesture context
  • Tab 2: P0-only view with critical path callout block
  • Tab 3: assign flow — assignee picker, priority, target date, notes, Cancel + Assign Now
  • Tab 4: resolved list — green card style, P0/P1/P2 chip, date, resolver, release version
  • Filter strip: All / P0 / P1 / P2 / Unassigned / Blocked
