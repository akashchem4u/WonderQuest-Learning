Component Spec: Teacher Assignment Status Board v2
  • Desktop 900px+ layout with responsive table and sticky nav bar (bg #2a5080)
  • Filter chips: All / Active / Completed / Draft / Overdue — each shows count, color-coded
  • Table columns: name, skill+band, assigned date, learner count, completion bar, due date, status chip, actions
  • Status chips: Active (#1a7040), Completed (#506070), Draft (#6030a0), Overdue (#9a2a2a)
  • Band chips: P0 red-tint, P1 amber-tint, P2 green-tint, P3 purple-tint
  • Actions: View (opens Tab 2 detail), Edit (opens form), Resend (overdue only)
  • Stats bar: 4 KPI cards at top — Active, Avg Completion, Overdue, Completed This Week
  • Tab 2: two-panel detail — assignment info left, completion summary right, question breakdown, learners not complete
  • Tab 3: per-learner completion list with avatar, name, progress note, status chip; exportable CSV
  • Tab 4: create-assignment form — name, band, skill, question count, learner group, due date; Save Draft / Assign Now
  • Contact-safe: no PII beyond first name + last initial shown, no contact info surfaced
