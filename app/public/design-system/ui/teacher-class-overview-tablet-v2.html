teacher-class-overview-tablet-v2 — Specification

Persona: Teacher · Tablet 768px · Font: Sora · Background: #f0f4f8 · Nav: #2a5080

API Endpoint: GET /teacher/class-overview returns { stats{}, sns_distribution{}, support_queue[], assignments[], at_risk[] }

  • SNS tiers labeled: Opportunity / Growing / Strength — NEVER raw % scores
  • Support queue sorted: flag (red left-border) then watch (amber) then monitor (green)
  • Assignment completion shown as count/total (e.g. 21/24), not percentage
  • At-risk grid: 3 columns, quick-action buttons per card
  • No grade labels, no missed-day shaming language, no accuracy % displayed anywhere
  • Empty queue state: "All students are on track today" (no negative framing)
  • Tablet-optimized: 768px max-width, flex/grid layouts wrap at smaller sizes
