Parent Skill Compare Mobile — Spec

Access points

  • From family summary → tap skill name when multiple children share the skill
  • From child skill detail → "See how Leo is doing on this skill →" CTA (only shown if sibling shares the skill)
  • From "All Skills" overview row when multiple avatar dots are shown

Same-band vs different-band rules

ConditionDisplay
Both children in same band, same skillSide-by-side compare cards with vs/+ divider
Children in different bands, analogous skill categoryTwo separate skill cards with band labels; callout explaining different levels
Only one child has this skill (other hasn't reached it yet)One child card + "Leo hasn't reached this skill yet" placeholder

Insight generation rules

ScenarioInsight shown
One support + one strength"[Strength child] knows this well — try practicing together. Teaching is great reinforcement!"
Both strength"Both kids are strong here! Keep celebrating this." (green callout)
Both support"Both kids could use practice here — perfect for a family activity this week."
Both next/approaching"Both are close — a little push this week could unlock it for both!"

Activity tagging rules

  • Activities with both children: tag both avatars + "Do together" section header
  • Single-child activities: tag only that child's avatar pill
  • Max 1 shared activity + 1 per child = 3 total maximum
  • Shared activities always shown first (most actionable for family)

Language rules (same as all parent screens)

  • Never show accuracy %, never compare scores between children
  • Never frame one child as "better" — use tier language only
  • When one child is stronger: frame as teaching opportunity, not ranking
  • No "behind" / "struggling" / "falling behind" language
