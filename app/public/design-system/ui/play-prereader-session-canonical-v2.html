📋 Pre-Reader Session Spec
Routes / API
EndpointPurpose
GET /session/startStart session, returns first question payload
POST /session/answerSubmit answer; returns next question or session-end
GET /session/hintRequest hint for current question
POST /session/endEnd session; returns XP delta, stars earned, world state
Reading Load Rules (Pre-Reader)
Max question text: 8 words. Emoji stimulus required on every question.
Answer labels: ≤ 2 words each. Single noun preferred (e.g. "Cat" not "A cat").
No grade labels displayed anywhere in UI. Use world names only.
No skill IDs, node IDs visible to child. Internal only.
No timer displayed. No countdown pressure.
Wrong answer: Never show score deduction. Never say "wrong". Use "Oops!" or "Let's try again!"
Answer Card Layout Rules
CountGridCard height

Voice Coach Behavior
Active indicator: Pulse ring on 🔊 button in world-header whenever TTS is playing or available.
Auto-read: Question text reads aloud on load (500ms delay). Answer labels read on card focus.
Wrong-answer: Coach appears automatically; delivers hint in 1–2 short sentences.
Session end: Coach delivers encouragement phrase.
