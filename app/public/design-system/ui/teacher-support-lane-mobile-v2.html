Spec
Component Spec: Support Lane Mobile v2
  • 390px mobile, Sora, bg #f0f4f8, nav #2a5080
  • Tab 1: overview list split by tier — Flag (#9a2a2a) and Watch (#c07800) sections
  • Tab 2: single-learner focus — metrics row, recent sessions, active interventions
  • Tab 3: skill snapshot — per-skill progress bars, trend arrow, P0/P1 band labels
  • Tab 4: append-only intervention timeline — timestamped, color-coded by entry type
  • Fixed quick-actions bar on focus screens: Note / Assign / Message
  • Contact-safe: all parent messages route through WonderQuest only — no phone/email PII
  • Intervention log is read-only (append-only) — no edit or delete actions shown
  • Entry types: Observation (blue), Concern (red), Assignment (amber), Parent contact (green)
