Child Safety Rules
No reading walls: Badge descriptions ≤ 12 words. Emoji-first always.
No grade labels: Use world names + emoji only.
No accuracy %: Never show "you got 60% right". Use stars and XP.
Wrong answer: Never surfaces in badge detail. Positive framing only.
Locked badge naming: Show friendly name + lock icon. No "Failed" or "Not good enough" language.
