Rules

No timers on shape scenes. Hint available from first attempt. Outline pulse uses CSS keyframes on SVG drop-shadow. Trace hint uses stroke-dashoffset animation. 3 stars always awarded for correct answer regardless of attempt count. Voice coach always visible and reads prompt on load via TTS. Wrong tap = shape shake only, no text feedback, no star penalty.
