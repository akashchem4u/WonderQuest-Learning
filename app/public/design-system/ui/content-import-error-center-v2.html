How to fix: No Correct Answer

  1. Open the source CSV and find the affected row number.
  2. Locate the correct_answer column (column G).
  3. Set exactly one option row to true.
  4. Ensure no other option in the same question group is also set to true.
  5. Re-import the corrected CSV.

Import Error Center spec. This surface handles all post-import validation failures, providing triage, auto-fix, and export tools.

Purpose

After a CSV batch is imported, all validation errors are grouped by type and surfaced here. Content admins can triage, auto-fix eligible errors, and download error reports.

Hard Blocks

  • No Correct Answer: question has 0 options marked correct
  • Duplicate Options: two+ options with identical text in same question

Hard blocks prevent the entire batch from publishing until resolved.

Soft Errors

  • Invalid Skill ID: maps to unknown node — auto-map suggested if confidence ≥ 85%
  • Malformed CSV: structural errors — manual fix in source CSV required

Auto-Fix Rules

  • Duplicate options: append " (2)", " (3)" to disambiguate
  • Invalid skill IDs: map to highest-confidence match ≥ 85%
  • All auto-fixes are logged and reversible per question ID

CSV Export

Download errors as CSV includes: row_number, error_type, error_detail, auto_fix_available, suggested_fix.
