Release Readiness Formula

Score = P0 Feedback (25 pts) + Routes (25 pts) + Content Coverage (20 pts) + QA Sign-off (20 pts) + Owner Sign-off (10 pts)

GO threshold: ≥ 90 / 100

Alert Escalation Rules

  • P0 banner pulses (CSS animation) while active
  • P1 unresolved at 24 hours → SLA breach, escalate
  • P2 unresolved at 72 hours → SLA breach flag
  • Route 500 sustained >5 min → auto-create P0 incident
