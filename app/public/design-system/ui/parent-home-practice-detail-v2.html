Tab 5 — Spec Notes
Route, content generation rules, practice moment caps, language rules (NEVER accuracy/missed sessions), teacher note display rules, weekend activity rotation, and design tokens.
Route
Route: /parent/child/:childId/practice. Accessible from: Family Summary "Try at Home" CTA, child detail screen, bottom nav Practice tab. Always shows one selected child — no multi-child view on this screen.
Practice Moment Generation Rules
RuleDetail
Count2–3 moments max. Support-tier skills shown first; Strength-tier second; Next-tier third.
Refresh cadenceNew suggestions every Monday. Parent can tap "Refresh" for a new set once per day.
Content toneNEVER "Mia is struggling with". ALWAYS "Mia is building toward / working on / developing".
DurationAll activities: 2–20 min max. Weekend cards: 15–30 min. Never suggest multi-hour commitments.
TagsAlways show: duration + skill name. Optional: activity type (Mealtime / Active / Creative / Game / Errand).
Done stateParent marks as done. Done cards shown muted at bottom. "Check back next week" message shown when all done.
Language Rules (Parent-Facing)
Never saySay instead
Struggling with fractionsBuilding toward fractions / Working on fractions
Missed X sessions this weekHas been busy this week / Check back soon
Scored 60% on fractions(Never show scores)
Needs to practice moreAny extra practice at home makes a big difference
Behind on skillDeveloping this skill right now
Teacher Note Rules
Show max 1 recent note (last 7 days) prominently. If note older than 7 days, show collapsed with "earlier note from Ms. Chen" disclosure. Thread shows most recent first. If no note: show the "Ideas on their way" placeholder — never show an empty note section. Teacher notes sourced from intervention timeline notes tagged as parent-visible.
Weekend Activity Rules
Weekend cards appear from Friday through Sunday. Removed Monday morning and replaced with weekday suggestions. Max 2 weekend cards. Rotate weekly — same activity cannot appear two weeks in a row for the same child. Weekend cards are longer format (15–30 min) and more elaborate than weekday moments.
