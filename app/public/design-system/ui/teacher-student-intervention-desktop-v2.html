teacher-student-intervention-desktop-v2 — Specification

Persona: Teacher · Desktop 1024px+ · Sora font · #f0f4f8 bg · #2a5080 nav · 3-column layout

API: GET /teacher/student/{id} → full student profile, sessions[], intervention_plan, notes[], assignments[]

API: POST /teacher/intervention/save { student_id, plan_text, type_tags[] } → 201 Created

API: POST /teacher/notes/append { student_id, note_text } → append-only, no delete endpoint

  • Left col: student card (avatar, name, tier badge, streak, XP) + recent sessions list
  • Center col: 8-week skill trend SVG bars + intervention plan textarea + type chips + save button
  • Right col: notes timeline (append-only, dot + note + date) + assignment links
  • Tier badge colors: Opportunity=amber, Growing=blue, Strength=green
  • SNS labels: Opportunity / Growing / Strength — NEVER accuracy %, NEVER missed-day language
  • Notes are append-only by design — no edit or delete UI exposed to teacher
