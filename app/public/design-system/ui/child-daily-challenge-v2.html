Engineering Spec · WonderQuest Child Daily Challenge v2

API Routes

  • GET /challenge/today — returns today's challenge for the child
  • POST /challenge/answer — body: { challengeId, questionIndex, answerId }
  • POST /challenge/skip — body: { challengeId }; streak preserved
  • GET /challenge/history — returns array of past completions

Challenge Format

  • 3 questions per daily challenge
  • Mixed skill types: matching, picture choice, sound/word link
  • Pre-reader: ≤8 words per question; K-1: ≤15 words
  • 2×2 answer grid (emoji + short label), no text-only answers

Reward Table

  • All 3 correct: +3 ⭐ · +50 XP · streak +1
  • Partial (1–2 correct): +1 ⭐ · +20 XP · streak preserved
  • Skip: +0 ⭐ · +0 XP · streak preserved (no penalty)
  • Miss (no open before midnight): streak resets to 0

Streak-Safe Rules

  • Skip does NOT break streak — child can come back same day
  • Partial completion (any answer submitted) counts as "opened"
  • Streak breaks only on a full miss: no challenge opened before local midnight
  • UI must never say "you'll lose your streak" — only positive nudge allowed

Child Safety Rules

  • No timers shown anywhere on challenge surfaces
  • No shame language on wrong answer — gentle dim + shake only
  • No wrong-answer counter visible to child
  • Skip framing: "saving your quest" (neutral/positive)
  • No grade labels, no skill IDs, no score percentage shown
  • Wrong answer: no star deduction, no "X" icon, no red text
