Design spec for play-world-backdrop-rotation-v2.

Component Identity

Fileplay-world-backdrop-rotation-v2.html
AudienceChild (ages 4–8)
Width375px mobile

Worlds

Dream WorldPurple star gradient + twinkling particles
Ocean WorldDeep blue + animated waves + bubbles
Fire WorldWarm orange radial + rising embers
Star WorldNight sky + shooting stars + gold accents

Rules

• All backgrounds use pure CSS — no image assets required.
• Question card uses backdrop-filter blur overlay for readability on any background.
• Particle/animation effects use CSS @keyframes only.
• World unlocked by child progression; not all worlds available from start.
• backdrop-filter may need -webkit- prefix for Safari.
