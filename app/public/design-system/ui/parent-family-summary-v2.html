Panel 5 — Spec Notes: Canonical spec for /parent (family summary view). This is the parent dashboard home — the first thing a parent sees on login. Route: /parent.

Parent Family Summary — Spec

Route: /parent — parent dashboard home. Shown on every login. Summarizes the current week across all children in the family.

Week hero card rules

ConditionHeadline toneChips shown
All children — 0 building skillsCelebratory — "Great week! Everything is looking strong."Strengths count only
1+ children with building skillsEncouraging — "Great week — one skill to practice together."Strengths + building count
No sessions this weekGentle nudge — "No activity this week — pick up where you left off!"None
First week (new family)Welcome — "Maya is all set up! First adventure is waiting."None

Child card SNS trio

LabelColorDefinition
Strengths#58e8c1 / #0d7a5fSkills in Strength tier this week
Building#ff7b6b / #c0392bSkills in Support tier (needs practice)
Almost there#5586e0 / #2255aaSkills approaching next milestone

Activity suggestions rules

RuleNotes
Show 2–3 activities maxToo many is overwhelming — prioritize Support-tier skills first
Rotate suggestions weeklySame activity should not appear 2 weeks in a row
Multi-child: interleaveAlternate child per activity row; flag sibling join opportunities
Duration + context requiredEvery activity shows "X min · [occasion]" — parents need to quickly assess feasibility

School messages rules

ConditionDisplay
Unread message <7 daysShow with red unread dot, border-left teal accent
Read message <14 daysShow without unread dot
No messagesShow "No messages from teachers yet" empty state

Multi-child layout rules

  • 1 child: full-expanded card (SNS + skill rows always visible)
  • 2 children: both expanded by default on mobile; desktop left rail shows both
  • 3+ children: collapsed by default on mobile (tap to expand); desktop left rail scrolls
  • Desktop: selected child drives center panel content

Language rules (same as all parent screens)

NEVER sayInstead say
Accuracy %, correct/wrong countsTier labels (strength/building/almost there)
"Missed X days" / attendanceNever surface attendance framing
"Behind" / "struggling" / "failing""Building this skill" / "could use practice"
Week comparison ("worse than last week")Always forward-looking framing only

Design tokens

TokenValueUsage

surface#ffffffCards, topbar, activity cards
accent#2c7a6eTeal — buttons, nav active, teacher accent

fontSoraAll parent-facing text
frame-width375pxCanonical mobile phone frame
