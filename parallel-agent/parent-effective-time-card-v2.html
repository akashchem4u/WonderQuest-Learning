ENG NOTE — parent-effective-time-card-v2.html
"Effective time" = time spent on questions where the child was actively engaged (not idle, not re-reading the same screen for >30s).
Formula: effective_time = total_session_time − idle_time − hint_read_overflow_time.
Parent framing: "Focused learning time" — never "efficiency %" or "wasted time". Always encouraging.
Accuracy % is NOT shown here — this card is about engagement, not correctness.
Shown in: parent weekly report (full card) and parent dashboard (compact pill). NOT on child-facing screens.

What does "focused time" mean?
This is the time Maya spent actively answering questions — not counting pauses, re-reads, or moments between questions. It's the most useful measure of how much active learning happened.
Component: parent-effective-time-card-v2.html
Formula: effective_time_seconds = session_duration_seconds − idle_seconds − hint_overflow_seconds
effective_pct = (effective_time / total_time) × 100, rounded to nearest integer.
Never show accuracy %, wrong counts, or raw idle counts to parent.
