📋 play-correct-state-v3 · Engineering Spec

Purpose

  • Standardized correct state for all play scenes — applied via CSS classes, not new components
  • Star counter updates in top bar with star-bump animation (scale 1 → 1.3 → 1, 0.4s spring)
  • Coach message is band-appropriate — see coach scripts table below
  • Star bonus chip appears with obj-pop animation, 0.1s delay after card turns green

Correct State Elements (in order)

  • 1. Star counter bumps (+1 visible number + scale animation)
  • 2. Chosen card gets .correct class → green border + bg + checkmark badge
  • 3. Unchosen cards get .dimmed class → 35% opacity, pointer-events none
  • 4. Coach bubble updates with celebration message
  • 5. Star bonus chip pops in below answer grid
  • 6. Bottom bar swaps to single CTA button (band-colored)

Band CTA Buttons

  • Pre-K Gold linear-gradient(135deg, #ffd166, #f0a000), dark text
  • K-1 Violet linear-gradient(135deg, #9b72ff, #7248e8), white text
  • G2-3 Mint linear-gradient(135deg, #58e8c1, #30b090), dark text
  • G4-5 Coral linear-gradient(135deg, #ff7b6b, #d04a38), white text

Coach Correct Scripts by Band

  • Pre-K: "Yes! [count]! You counted them all!" — concrete, simple, affirms counting strategy
  • K-1: "Buh-buh-bee! B is right! You heard it perfectly!" — phoneme reinforcement
  • G2-3: "You found it in the passage! [quote]" — evidence-citing reading strategy
  • G4-5: "You read between the lines! [clue list]" — metacognitive inference framing

Optional Overlay Mode

  • Triggered by a higher-value correct (e.g. boss node answer, streak milestone)
  • Full-screen overlay with burst ring animations (burst-expand keyframe), trophy-bob emoji, large "+1 Star" chip
  • Auto-dismisses after 1.2s timeout or on tap
  • NOT shown on every correct answer — only for special moments (configurable via game engine)
