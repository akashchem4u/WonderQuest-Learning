ENG NOTE — teacher-recent-wins-panel-v2.html
Recent wins panel — shows positive class-level achievements: skills mastered, streaks reached, band advancements, star milestones.
Placement: Right column of teacher home (desktop), below stats on mobile, in Class Overview analytics.
Win types: skill_mastered | streak_milestone | band_advancement | class_star_milestone | perfect_session_streak.
Privacy: Student first names only. Win events are POSITIVE by design — no negative signals here. This panel NEVER shows wrong answers, queue flags, or accuracy %.
Framing: Always celebratory. "Bella mastered Long Division!" not "Bella completed Long Division".
Deduplication: Max 1 win per student per event type per day (so a student who mastered 3 skills today shows as one combined entry, not 3).

Privacy

Student-level wins show first name only (FERPA). Class-level wins show count + skill name — no individual names. This panel NEVER shows: accuracy %, wrong answers, mastery score (only status transition), queue flags, negative signals.

Deduplication Rule

Max 1 win per student per event type per calendar day. If a student mastered 3 skills today, show as "Bella mastered 3 skills today!" (combined). This prevents feed flooding.
