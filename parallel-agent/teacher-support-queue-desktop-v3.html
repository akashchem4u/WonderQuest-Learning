Dedicated support queue page — full list of students needing teacher attention, with expanded detail cards.
Queue triggers (4 types):
  • ⚠️ Confidence floor: confidence_floor_hit ≥ 3 on same skill (amber)
  • 📅 Absence: no sessions ≥ 5 calendar days (amber)
  • 💡 Hint pattern: hint_requests ≥ 5 on same skill this week (amber)
  • 💙 Band ceiling: at band max, ready to advance (blue/positive)
Expanded card shows: trigger detail, skill context, suggested action — NOT wrong answers, NOT accuracy %, NOT specific question text.
Actions per item: View student / Resolve / Dismiss / Note (log teacher note).
Filters: by trigger type, by band, by status (new / acknowledged / resolved).
Privacy: first names only, pattern-level skill data, no parent contact info.

Actions per Queue Item

View student (→ student profile) | Log a note (freetext teacher note, timestamped) | Mark resolved (removes from queue, archived 30 days) | Dismiss (hides for 7 days, re-surfaces if trigger repeats).

Automatic Queue Management

Queue items are auto-resolved when: student has not hit the trigger condition in 5 days (confidence floor / hint pattern) | student resumes sessions (absence) | teacher marks resolved. Band ceiling is never auto-resolved — requires teacher action.
