ENG NOTE — content-question-gallery-v3.html
Visual card gallery of all questions, designed for bulk review and curriculum audits. Card-based grid with answer preview on selection.
Purpose: Used during curriculum review sprints to quickly scan all questions in a skill or subject. Visual scanning is faster than the list browser for quality checks.
View toggle: Gallery (4-col grid) ↔ List (table). Same filter/sort options in both views.
Card content: Question type badge · Status badge · Illustration placeholder (if asset) · Question text (2-line clamp) · Band tag · Impression count · Skip rate / correct rate mini-stats.
Selected card: Expands an inline detail panel below the grid row (not a separate pane). Shows full answer options, stats, quick actions.
Flag indicator: Red top bar on blocked cards. Amber top bar on under-review cards.

Component: content-question-gallery-v3.html
Visual question card gallery. 4-col grid. Gallery ↔ List view toggle. Red top bar (blocked) / amber top bar (review). Selected card expands inline detail. Flag indicator + stats per card.

Gallery vs Browser

Gallery (this component) is optimised for scanning all questions in a skill or subject at once — useful during curriculum review sprints. Browser (content-question-browser-desktop-v3.html) is optimised for drill-down on a specific question with full error report detail. Both show the same underlying data. Gallery is the default view for "review all questions in this skill"; Browser is the default for "find and fix a specific flagged question".

Card Flag Indicators

Blocked cards have a 3px red top bar (derived from .qgc-flag-bar). Under Review cards have an amber top bar (.qgc-review-bar). Live cards have no top bar. Asset 404 cards show a "404" badge in the visual area instead of the question image. These visual indicators allow rapid scanning without reading the status badge on each card.

Selected Card Expansion

Clicking a card shows an inline expansion panel above the gallery row containing the selected card (not a modal). The panel shows full answer options (with correct/wrong highlights), stats, and quick action buttons. This avoids losing context of the gallery while reviewing a question. The panel closes when another card is selected or when Escape is pressed.
