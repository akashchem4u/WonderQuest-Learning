ENG NOTE — teacher-home-recommendation-card-v2.html
System-generated recommendations for teacher action on home screen.
Placement: Right column on teacher home (desktop), below stat row on tablet/mobile. Part of the teacher dashboard right-rail alongside Watchlist and Recent Wins.
Rec types: band_advancement (N students ready) | support_needed (struggling pattern detected) | skill_assign (suggest assign next skill) | streak_at_risk (engagement dropping) | class_milestone (celebratory).
Priority ordering: Support > Advancement > Skill suggestion > Milestone. Max 4 recs shown at once.
Dismissal: Teacher can dismiss individual recs. Dismissed recs won't reappear for 7 days unless trigger re-fires.
Not alarmist: Recs use neutral/positive framing. No "failing", "at risk", "urgent". Instead: "may need support", "ready to advance", "keeping an eye on".

Framing Rules

All recs use positive/neutral language. NEVER use: "failing", "at risk", "urgent", "falling behind", "not meeting expectations". Instead use: "may need support", "ready to advance", "worth keeping an eye on". Support-type recs always link to the existing queue item — never duplicate the queue entry.

Priority Order

1. support_needed → 2. band_advancement → 3. skill_assign / streak_at_risk → 4. class_milestone. Maximum 4 shown. Items beyond 4 are accessible via "See all".
