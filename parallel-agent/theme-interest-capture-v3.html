Rules & Constraints

  • Minimum 1 interest tag required to proceed; maximum 12 tags per profile
  • Version number increments on every update; full history retained in child_interest_profiles
  • Interest data is never surfaced in teacher or owner views
  • Interest capture is optional for returning users during re-onboarding
  • COPPA compliance: parent-only control over all interest data; no child-facing data entry
  • Interests are used exclusively for content personalization; never shared externally
  • Child identifier: first name only; no date of birth, no surname, no unique identifiers beyond child_id FK
  • UX: adventure/positive framing only; no academic language, no timers, no accuracy metrics
