📋 play-grade45-math-scene-v2 · Engineering Spec

Purpose

  • G4–5 multi-step math scene — word problems requiring 2 operations (e.g. multiply then add)
  • Step breakdown panel makes problem structure explicit — metacognitive scaffolding
  • Coral (#ff7b6b) G4–5 theme. No timer, no accuracy %, stars never decrease.

Tabs / States

  • Active Word problem card + 2-step breakdown (both unknown), 4 answer buttons
  • Correct Both step rows get checkmarks + filled answers, correct button → green, coral CTA
  • Wrong Wrong button shakes, step 1 hint revealed (answer shown), step 2 still unknown — targeted scaffold
  • Spec This document

Step Breakdown Panel

  • 2 rows: step number circle + step description text + result (or "?" + pulse)
  • Active: both show "= ?" with coral-pulse animation
  • After wrong: step 1 auto-revealed with answer — step 2 remains open for child to solve
  • On correct: both rows get green checkmarks + filled results

Math Scope (G4–5)

  • Two-step word problems: ×+, ×−, ÷+
  • Multi-digit multiplication: 2×2 digit (24 × 3)
  • Fractions: simple equivalents, comparing (½ vs ¾)
  • Division with remainders
  • Answer choices: 4 plausible values with common error traps (e.g. doing only step 1)

Wrong Answer Scaffold

  • If child answered 24 (Step 1 only): hint says "You got step 1! Now add 7 more."
  • If child answered 38 (added 7 to wrong number): hint reveals step 1 correct answer
  • Wrong message always points to the specific step that needs revisiting

Design Tokens (G4–5 Coral Theme)

  • Coral: #ff7b6b — progress segs, step numbers, problem highlights, CTA
  • CTA: linear-gradient(135deg, #ff7b6b, #d04a38), white text
  • Phone frame shadow: 0 0 0 2px #ff7b6b22
  • Correct step: #50e890 checkmark + result
  • Unknown step pulse: coral-pulse animation (opacity flicker)
  • Star-safe badge: always shown on active + wrong states
