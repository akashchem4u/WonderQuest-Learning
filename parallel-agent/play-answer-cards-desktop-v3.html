📋 play-answer-cards-desktop-v3 · Engineering Spec

Component Overview

  • Shared answer card component used across all play scenes — desktop and tablet sizes
  • Base class: .ac — applies background, border, border-radius, flex layout, transition
  • Size modifier class applied per use case (see table below)
  • State class applied via JS when answer is submitted

Size Classes

ClassMin HeightUse Case
.ac-standard100pxStandard 4-choice 2×2 grid
.ac-large120px2-choice layout (bigger target)
.ac-compact72pxFull-sentence answer list (reading)
.ac-wideautoFull-width row, flex-direction: row — text aligned left
.ac-letter80×80pxSingle letter + phoneme label (phonics)
.ac-numeral70×70pxSingle number answer (math)

State Classes (JS-applied)

ClassVisualBehavior
.correctGreen border + bg + ✓ badgecursor: default, no hover lift
.wrongRed border + bg + shake animationshake once, cursor: default; others remain active
.dimmed35% opacitypointer-events: none — shown after correct answer
.active-choiceViolet border + tintFor multi-select or previewing sequences

Band Accent Classes

  • .band-prek on parent — hover border color → gold (#ffd166)
  • .band-k1 (default, no class needed) — hover border → violet (#9b72ff)
  • .band-g23 on parent — hover border → mint (#58e8c1)
  • .band-g45 on parent — hover border → coral (#ff7b6b)

Correct State Checkmark Badge

  • CSS ::after pseudo-element — no extra DOM needed
  • Content: "✓" at top-right corner (position absolute)
  • Color: var(--correct) = #50e890

Wrong + Shake Animation

  • Class .wrong applies ac-shake keyframe (translateX ±8px, 0.5s, ease-out)
  • After animation completes: card stays red — not reset to neutral (keeps history visible)
  • Other cards remain interactive — child retries from remaining choices

Layout Patterns

  • 2×2 grid: display: grid; grid-template-columns: 1fr 1fr; gap: 12px;
  • 4-up row: display: flex; gap: 10px; with .ac-letter or .ac-numeral
  • Full-width list: display: flex; flex-direction: column; gap: 10px; with .ac-wide
  • 1×2 wide: display: grid; grid-template-columns: 1fr 1fr; gap: 14px; with .ac-large
