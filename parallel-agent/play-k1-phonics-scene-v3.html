📋 play-k1-phonics-scene-v3 · Engineering Spec

Purpose

  • K-1 phonics beginning-sound identification scene — child sees a picture/word and must tap the letter that makes the starting sound
  • Violet (#9b72ff) K-1 theme throughout — distinct from Pre-K gold
  • No timer, no accuracy %, stars never decrease

Tabs / States

  • Active Emoji hero with word label, 2×2 letter choice grid (letter + phoneme label below), bottom 3 action buttons + star-safe
  • Correct Hero → green, correct letter btn gets selected-correct, others dimmed, violet CTA "Next Sound ✨ +1⭐"
  • Wrong Wrong btn gets selected-wrong + shake, red banner with physical cue ("Say 'bee' out loud"), other choices remain active
  • Spec This document

Image Hero

  • 160×160px card, border-radius: 20px, 3px violet border
  • Large emoji (72px) + word label below in muted color
  • violet-glow animation: box-shadow 2.5s ease-in-out infinite
  • On correct: border → #50e890, bg → #0d2a1a, word label turns green

Letter Choice Buttons

  • 2×2 grid, 88px tall, Nunito 900, 2.8rem letter + 0.72rem phoneme label
  • Phoneme label provides audio scaffold for hesitant readers: "buh" / "sss" / "duh" / "rrr"
  • obj-pop staggered entry animation 0.05s per button
  • Hover: violet border + -2px lift + violet letter color
  • Wrong: shake + red border, other choices remain active for retry
  • Correct: green border, others dimmed to 0.35 opacity

Coach Script Examples

  • Active: "Buh! Buh! What letter makes that sound? Listen to the beginning!"
  • Correct: "Buh-buh-bee! B is right! Bee starts with the B sound — you heard it perfectly!"
  • Wrong: "Not quite — say 'bee' out loud. What's the very first sound that comes out?"

Phoneme Label Convention

  • Show the raw phoneme below each letter button: B="buh", S="sss", D="duh", R="rrr"
  • This provides audio scaffolding without giving the answer away
  • Letters grouped to avoid similar-sounding distractors at K-1 level

Progress Strip

  • Segment bars, flex:1, 6px height, violet theme: done=#9b72ff, active=rgba(155,114,255,0.6)

Design Tokens (K-1 Violet Theme)

  • Violet accent: #9b72ff — progress segs, hero border, CTA
  • CTA: linear-gradient(135deg, #9b72ff, #7248e8), white text
  • Topbar: standard dark #1a1440 card bg (not gold gradient)
  • Correct: #50e890 / #0d2a1a bg
  • Wrong: #ff7b6b / #2a0e0e bg
  • Star-safe badge: always visible on active + wrong states
