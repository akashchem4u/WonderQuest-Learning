📋 play-voice-replay-controls-v3 · Engineering Spec

Replay Button

  • Default: background: #2a2050; color: #9b72ff; border: 1px solid #3a3060
  • Playing: adds .playing class → green color, pulse ring animation, label → "Playing…" + wave bars
  • Playing state dims (but does NOT disable) Hint and IDK buttons — child can still tap
  • Replay is NEVER disabled — always accessible even mid-playback (tapping restarts audio)

Slow Mode Button

  • Hidden by default. Appears after child taps Replay 2+ times (adaptive behavior).
  • Also shown if child has "Slow" speed preference set in voice preferences
  • Toggle: off = rgba(88,232,193,0.08) bg; on = rgba(88,232,193,0.18) bg + mint border + box-shadow
  • When slow ON: all subsequent replays play at 0.75× TTS speed until toggled off

Adaptive Slow Suggestion

  • After 2nd consecutive replay tap without answering: show slide-in mint banner "Want to hear it more slowly?"
  • Shows the Slow button if not already visible
  • Banner dismisses on any answer tap or scroll

Re-read vs Replay

  • 🔊 Replay: audio scenes (Pre-K sound, phonics, K-1 all scenes) — re-speaks the question
  • 📖 Re-read: text-primary reading scenes (G2-3, G4-5) — scrolls passage into view + flashes highlight
  • Same button position, same flex:1 sizing — only label + color changes by band

Button Layout Rules

  • Standard: 3 buttons, flex:1 each — Replay / Hint / IDK
  • Pre-K: 2 buttons — Replay (flex:2) / IDK (flex:1) — Hint removed at Pre-K (too abstract)
  • With slow: 4 buttons — Replay / Slow / Hint / IDK (each flex:1, smaller font if needed)
