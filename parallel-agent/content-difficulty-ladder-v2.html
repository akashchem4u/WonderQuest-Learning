ENG NOTE — Difficulty Ladder v2: Difficulty levels 1–5 per skill: 1=Introductory / 2=Foundation / 3=Core / 4=Advanced / 5=Extension.
Each skill has a difficulty_level field (1–5). Adaptive engine serves lower difficulty first within a skill path.
Band constraints: P0 Pre-K should have no level 4–5 skills. P1 K–1: max level 4. P2/P3: all levels allowed.
Gap rule: If a subject has a gap at level 3 but level 4+ skills exist, adaptive path is broken — flag as "broken ladder".
