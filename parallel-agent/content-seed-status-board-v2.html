Seed pipeline stages: Assigned (to writer) → Scripted (first draft complete) → Curriculum Review → QA (safe language + fact check) → Ready (enters main review queue).
Each seed topic represents 1 new skill + 4–8 questions + 1 explainer + audio brief.
SLA per stage: Assigned→Scripted = 3 biz days; Scripted→Reviewed = 2 biz days; Reviewed→QA = 1 day; QA→Ready = 1 day.
Seed items overdue = assigned_date + cumulative_stage_sla < NOW(). Overdue = red border.
When stage = Ready, item auto-moves into the main Review Queue (content-review-queue-v2).
