Business Rules
  • Theme is cosmetic only; it never affects content difficulty, band placement, or skill sequencing.
  • Child cannot change their own theme family — parent control only.
  • Pack within a family is auto-selected by best interest-tag match between child profile and pack tags.
  • Every theme switch is logged to child_theme_assignments for audit trail.
  • Theme change takes effect on the child's next quest session, not mid-session.
  • COPPA compliance: parent controls all cosmetic preferences; child first name only used in UI copy.
  • Band accents P0–P3 are independent of theme cosmetics; bands are never themed.
