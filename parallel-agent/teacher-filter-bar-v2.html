ENG NOTE — teacher-filter-bar-v2.html
Reusable filter bar component — used across student list, support queue, and analytics tables in teacher portal.
Filter dimensions: Band (P0/P1/P2/P3) | Activity (Active today / Needs check-in / Absent / All) | Status (All / In queue / Band ceiling) | Sort (Stars / Sessions / Streak / Name / Band).
Variants: Desktop inline bar | Mobile horizontal chip scroll | Dropdown filter panel (triggered by "Filter" button).
Active filter summary: Shows applied filters as removable chips below bar.
Not a privacy concern — filter dimensions are teacher-visible data dimensions only.

Variants by Viewport

ViewportPattern
Desktop (≥1025px)Full inline chip bar (all dimensions visible at once)
Tablet (768–1024px)"Filter" button → dropdown panel; quick band chips visible inline
Mobile (≤767px)"Filter" button + horizontal scroll chip row (band only inline; rest in panel)

Active Filter Summary Bar

When any non-default filters are active, show a summary chips row below the filter bar. Each chip shows the filter label with an ✕ to remove. "Clear all" link on right. Summary bar disappears when all filters return to default.
