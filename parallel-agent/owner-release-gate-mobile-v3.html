Same gate logic as desktop. Mobile layout: collapsed category rows, pinned launch button, full-screen confirmation overlay on launch tap.

Mobile Launch Safety

On mobile, tapping the (active) launch button shows a full-screen confirmation overlay: version number displayed prominently, brief changelog summary, "Confirm deployment" + "Cancel" buttons. "Confirm" requires a second tap with brief 1s delay (anti-accidental-tap). This is intentionally harder to trigger than desktop (which uses a typed version number).
