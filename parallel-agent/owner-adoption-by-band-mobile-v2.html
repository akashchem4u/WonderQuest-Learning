Component: owner-adoption-by-band-mobile-v2.html
Mobile adoption by band. Same data as desktop — vertical band cards + compact advancement bars. No stacked chart on mobile.

Layout Differences from Desktop

Desktop: band cards in 2×2 grid + stacked MAU chart + full retention table. Mobile: single-column band cards with inline metrics + compact advancement section. Stacked chart replaced by per-band % share bar within each card.

Navigation

Reached from More tab → Analytics → Adoption by Band. Distinct from Command Center (which is the main analytics page). Band adoption is one level deeper than command center.
