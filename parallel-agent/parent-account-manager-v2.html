Account Deletion Flow

Step 1: Click "Delete account" → confirmation dialog with 90-day data retention explanation.

Step 2: Checkbox ("I understand…") must be checked. Password re-entry required. Then destructive button becomes active.

After deletion: Account enters 90-day grace period. Email sent to parent with reactivation link. After 90 days, all data purged (parent + child). Child progress data follows same 90-day window.

COPPA

Parent's own profile data has standard adult privacy rights (GDPR/CCPA). Child profiles are covered by COPPA — children's data export is included in the parent's export ZIP and follows the same 90-day deletion window.
