ENG NOTE — teacher-mobile-route-shell-v2.html
Teacher mobile navigation shell — defines the chrome (header, bottom tabs, side drawer) for all teacher mobile routes.
Bottom tab bar (5 items): Home | Students | Queue | Messages | More
Header variants: Main header (title + icons) for root routes. Back header (back arrow + title) for child routes.
Side drawer (More): Opens from More tab. Contains: Assignments, Watchlist, Analytics, Feedback, Settings, Sign out.
Badge system: Queue badge = open queue item count. Messages badge = unread message count. Both shown on tab icons and in drawer.
Safe area: Bottom tab bar respects iOS safe area (home indicator). Add padding-bottom: env(safe-area-inset-bottom) on mobile.
Back navigation: Stack-based. Back header shown on: student detail, skill drilldown, intervention detail, assignment detail.
