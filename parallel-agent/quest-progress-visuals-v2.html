quest-progress-visuals-v2.html — Quest progress visual components. A quest = a skill path with N questions/challenges. Progress tracked as: stars earned (0–3 per level), questions completed, XP accumulated, streak. Visual components: (1) Star Rating (0–3 stars per question/level), (2) Quest Progress Path (node map showing completed/active/locked nodes), (3) XP Progress Bar (total XP with level milestones), (4) Streak Flame (consecutive-day streak). No timer anywhere. No accuracy %. Stars are purely celebratory — never used punitively.
