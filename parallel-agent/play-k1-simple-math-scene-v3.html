📋 play-k1-simple-math-scene-v3 · Engineering Spec

Purpose

  • K-1 simple addition scene — child counts two visible groups of objects and taps the total
  • Visual object groups reinforce counting-on strategy (not memorization)
  • Violet K-1 theme. No timer, no accuracy %, stars never decrease.

Tabs / States

  • Active Equation card with 2 emoji groups + count labels + "?" answer slot, 4 number buttons below
  • Correct Objects get highlight-obj filter, answer slot shows numeral, correct button → green, violet CTA
  • Wrong Wrong button shakes + red border, wrong banner with touch-counting prompt, star-safe shown
  • Spec This document

Equation Display

  • Flexrow: [group A emoji objects + count] + [op symbol] + [group B emoji objects + count] + [=] + [? or answer]
  • Each object gets staggered obj-pop animation (0.04s per emoji)
  • Count labels in violet below each group (e.g. "3", "2")
  • "?" pulses with violet-pulse keyframe while awaiting answer
  • On correct: "?" replaced with numeral in #50e890, objects get highlight-obj drop-shadow filter

Number Answer Buttons

  • 4 buttons, 70px square, Nunito 900, 2rem font
  • 4 adjacent numbers (e.g. 4/5/6/7 for answer 5) — distractors ±1 and ±2
  • Wrong: shake + red border, other choices remain active
  • Correct: green border, others dimmed

K-1 Math Scope

  • Addition only — sums 2–10
  • Max 5 objects per group (visual countability constraint)
  • Use emoji objects matching child's active theme (stars, animals, food, etc.)
  • No subtraction at this level

Coach Script

  • Active: "Count all the stars together! Touch each one as you count — how many in all?"
  • Correct: "3 + 2 = 5! 3… 4… 5! Perfect counting — you're a math star!"
  • Wrong: "Not quite — let's count again. Touch each star one at a time: 1, 2, 3… keep going!"

Design Tokens (K-1 Violet)

  • Violet: #9b72ff — count labels, eq-question pulse, progress segs, CTA
  • Correct: #50e890 — answer reveal + highlight-obj filter glow
  • Wrong: #ff7b6b shake + banner
  • Star-safe badge on active + wrong states
