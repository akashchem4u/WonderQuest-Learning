Technical Specification — Item 262

Preview Trigger

  • Web Speech API speechSynthesis.speak(utterance) — no server call made.
  • Any active utterance is cancelled first via speechSynthesis.cancel() before a new one starts.

Voice Mapping

Luna the Explorer tts_voice_key: "en-GB-female-soft" — closest match from getVoices(); fallback to first available.
Orbit the Robot tts_voice_key: "en-US-male-neutral"
Rex the Dino tts_voice_key: "en-US-male-deep"
Maya the Fairy tts_voice_key: "en-GB-female-light"
Zara the Scientisttts_voice_key: "en-US-female-clear"

Locale & Rate

Locale Respects child_audio_prefs.voice_locale; defaults to en-GB.
Normal rate utterance.rate = 0.9
Slow rate utterance.rate = 0.75

Privacy

  • No audio recorded.
  • No server call made during preview.
  • No voice biometrics collected.

Sample Phrases CMS

  • Stored in voice_sample_phrases(band_id, category, text) CMS table.
  • Updateable without a deploy — fetched at runtime.

WCAG Accessibility

  • [▶ Play] button has aria-label="Play sample for {VoiceName}".
  • Waveform bars are decorative — marked aria-hidden="true".
  • Playing state announced via aria-live="polite" region.
  • Band and category buttons expose aria-pressed toggle state.

Band Vocabulary

P0 Early learner — simple picture-based prompts.
P1 Phonics / early reading — rhyme and sound-based language.
P2 Fluent reader — conceptual vocabulary, shape and math language.
P3 Advanced — passage comprehension, abstract ideas.
