Navigation Behaviour (disabled nav items)

When no child is linked, dashboard nav items that require a child context are disabled (visual opacity reduction). Tap on disabled items shows a tooltip: "Link a child first to see [Report/Badges/etc.]". Items that stay active: Home, Settings.

Nav itemState when no child linked
HomeActive — shows empty state
ReportDisabled (40% opacity)
BadgesDisabled
NotificationsActive — may have account-level notifs
SettingsActive — account settings always available

Toast (Post-removal)

A dark (#1a1440) toast appears at the top of the main content area immediately after removal. Auto-dismisses after 3 seconds. Copy: "[Child name] removed · Progress saved. Re-link anytime." Toast is NOT a modal — user can interact with the page behind it.
