Component: parent-home-practice-planner-v3.html
Distinct from: parent-practice-planner-v2 (which is the in-app session schedule/limit settings).
This is the offline supplementary activity planner — curated tips tied to support areas.

How Activities are Selected

1. Fetch child's current support_areas (max 3 skills with mastery < 65).

2. Query activity_library WHERE skill_id IN (support_area_skill_ids) AND band_id = child.band_id ORDER BY rating DESC LIMIT 5.

3. Parent sees 2–5 suggestions; can save any to their weekly checklist.

Activity Library Requirements

RequirementDetail
Free / no materialsDefault filter. Activities requiring materials are tagged and shown last.
Short durationUnder 10 minutes by default; 15+ minute activities tagged "longer".
Positive framingNever "drill" or "test" — always "play", "explore", "discover".
Band-appropriateP0 Pre-K / P1 K–1 / P2 G2–3 / P3 G4–5 activity sets separately curated.
Skill-linkedEach activity maps to 1–2 skill IDs for filtering.

Checklist Tracking

Checklist is parent-only, manually checked. Items do NOT sync to child's in-app progress. No gamification on the parent side for checklist completion. Weekly summary note is celebratory, not a score.
