ENG NOTE — teacher-assignment-preview-v2.html
Teacher skill assignments — teacher manually assigns a specific skill as a priority focus for a student or the whole class.
Soft priority, not forced: WonderQuest adaptive engine considers teacher assignments as priority hints. The system will favour the assigned skill but may intersperse others. Teacher cannot force a locked linear path — adaptive engine always retains final say.
Assignment types: Class-wide (all students) | Group (filtered subset) | Individual (single student).
Due date: Optional. If set, system surfaces assignment more aggressively near due date.
Completion: Assignment auto-marks complete when student reaches mastery_status = Strong on assigned skill.
Max active assignments: 3 per student at once (to avoid overloading adaptive engine). Warn teacher if > 3.
