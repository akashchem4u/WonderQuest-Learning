📋 play-grade23-math-scene-v2 · Engineering Spec

Purpose

  • G2–3 math scene for multiplication (×2–×5 tables), two-digit addition, and subtraction
  • No object counting scaffolding — children encouraged to recall or use skip-counting strategy
  • Hint system reveals array (for multiplication) or skip-count strip (procedural scaffold)
  • Mint (#58e8c1) G2–3 theme. No timer, no accuracy %, stars never decrease.

Tabs / States

  • Active Large equation display, hint array shown (toggled by 💡 button), 4 answer buttons
  • Correct "?" replaced with answer numeral in green, button highlighted, mint CTA
  • Wrong Wrong button shakes + red, skip-count hint strip revealed automatically, star-safe shown
  • Spec This document

Equation Display

  • Flexrow: [left operand] [op symbol] [right operand] [=] [blank/answer]
  • Nunito 900, 3.5rem operands — large enough for at-a-glance recall
  • Blank (?) has mint-pulse animation and mint underline border
  • On correct: blank replaced with numeral in #50e890

Hint System

  • Array visual: 3 rows × 4 cols of mint dots (for 3×4). Toggled by 💡 Hint button. Hidden by default.
  • Skip-count strip: Revealed automatically after first wrong answer. Shows sequence (4, 8, 12, 16…) with target number highlighted in solid mint.
  • Both hints are cognitive scaffolds — they support strategy, not just give the answer

Answer Buttons

  • 4 buttons, flex:1 each, 68px tall, Nunito 900, 1.8rem
  • Distractors: ±3 adjacent plausible answers (e.g. 9, 12, 15, 16 for 3×4)
  • Wrong: shake + red border, other choices remain active for retry
  • Correct: green border, others dimmed

Math Scope (G2–3)

  • Multiplication: ×2, ×3, ×4, ×5 tables — products up to 25
  • Two-digit addition: 12+17, 24+38 (with/without carrying)
  • Subtraction: two-digit, no negative results
  • No division at this level

Coach Language (G2–3)

  • Active: "3 groups of 4! How many total? Recall it or skip count — you've got this!"
  • Correct: "3 × 4 = 12! 4, 8, 12 — three groups of four. Your multiplication brain is ON!"
  • Wrong: "Not quite — try skip counting by 4s. 4… 8… 12… count 3 jumps!"
  • More sophisticated framing than K-1 ("recall" / "skip count" / "groups of") — matches cognitive level

Design Tokens (G2–3 Mint Theme)

  • Mint: #58e8c1 — progress segs, equation blank, array dots, skip-count target, CTA
  • CTA: linear-gradient(135deg, #58e8c1, #30b090), text #0a2a22
  • Phone frame shadow: 0 0 0 2px #58e8c122
  • Correct: #50e890 / #0d2a1a bg
  • Star-safe badge: always shown on active + wrong states
