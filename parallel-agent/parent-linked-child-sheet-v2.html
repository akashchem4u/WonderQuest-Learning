Unlink Flow (2-step)

Step 1: Parent taps "Remove [name] from this account" red button → sheet transforms to confirmation sub-sheet with ⚠️ icon.

Step 2: Confirmation copy: "Progress data is saved and can be restored by re-linking with the same device." Two buttons: "Yes, remove [name]" (red, destructive) + "Cancel" (neutral). Cancel returns to default sheet.

After removal: Sheet dismisses → dashboard shows parent-linked-child-empty-v2 state → success toast for 3 seconds.
