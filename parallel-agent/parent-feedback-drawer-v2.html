Privacy Rules in Feedback Form

Email is auto-filled from parent account — parent can remove it to submit anonymously. Form never asks for child name or child details. If parent mentions child name in free text, that's their choice — not prompted.

Rating

Star rating is optional. 1–5 stars. Defaults to no selection. If rating ≤ 2, show "We're sorry to hear that — tell us more so we can improve" message below stars.
