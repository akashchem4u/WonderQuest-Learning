adult-topbar-system-v2.html — Top navigation bar component for adult personas. Height: 56px desktop / 52px mobile. Fixed top, full-width, z-index:200. Persona variants: Teacher (sky blue), Parent (violet), Owner (green). Topbar sections: Left=logo+wordmark+environment badge (owner only), Center=breadcrumb or page title (desktop) / page title only (mobile), Right=actions cluster (search, notifications, avatar menu). Notification bell: badge count from unread_notifications WHERE user_id=:id AND read_at IS NULL. Avatar menu: dropdown with profile/settings/help/sign-out.

Data Model & Derivations
-- Notification badge count SELECT COUNT(*) FROM notifications WHERE user_id = :id AND read_at IS NULL; -- Avatar initials derivation (JavaScript) const initials = (user) => (user.first_name[0] + user.last_name[0]).toUpperCase(); -- Breadcrumb: derived from URL path + page_titles map (not stored in DB) const PAGE_TITLES = { '/dashboard': 'Home', '/students': 'Students', '/students/:id': studentName, // dynamic '/progress/:childId': childName + "'s Progress", '/route-health': 'Route Health', // ... }; -- Child switcher (parent persona): session storage session.active_child_id → drives all data queries for session -- Environment badge: sourced from process.env.APP_ENV // Only rendered for owner persona // Values: 'production' → 'PROD' (red), 'staging' → 'STAGING' (amber), etc.
