Perfect Session Flag

"Perfect session" = child answered all questions correctly in that session for this skill. Shown as ⭐ gold badge in session log. NOT shown as "100%" — child language (stars) is preserved for parent too.

Support Tip Display

Tip card shown only if skill status = Building (mastery 40–64) or Just started (< 40). Strong skills (>= 65) show a celebration card instead. Same tip library as parent-home-practice-planner-v3.
