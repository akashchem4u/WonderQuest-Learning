ENG NOTE — owner-kpi-strip-desktop-v3.html
Persistent KPI strip at the top of every owner desktop console page. 8 key metrics with deltas and period context.
Metrics (left to right): MAU · DAU · MRR · ARR · Schools · Active Feedback · Release Gate · Route Health.
Active cell: Green underline on hover/selected. Red underline + red values for alert states (incidents, gate blocked).
Period selector: 7d / 30d / 90d / All Time. Deltas are always vs. previous equivalent period.
Clicking a KPI cell: Navigates to the corresponding detail page. Feedback → Feedback Workbench. Routes → Route Health. Gate → Release Gate.
Update frequency: Refreshed every 5 minutes (not real-time for cost reasons). Timestamp shown in period bar.

Persistent 8-cell KPI strip. Top of every owner desktop console page. Period selector (7d/30d/90d/All Time). Refresh 5 min normal, 10s on P0. Clickable cells.

Period Selector Behaviour

MAU and MRR are always monthly values — period selector does not change them (they show current month). Selecting 7d swaps the "Schools" cell to "Sessions" (more relevant for short periods). DAU average, Sessions, and Feedback counts respond to period selection. ARR is always annualised MRR × 12.

Refresh Rate

Normal: every 5 minutes (trade-off between freshness and API cost). P0 incident active: Route Health cell refreshes every 10 seconds; other cells remain at 5-minute refresh. Timestamp shown in period bar ("Updated N min ago").
