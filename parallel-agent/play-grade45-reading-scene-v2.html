📋 play-grade45-reading-scene-v2 · Engineering Spec

Purpose

  • G4–5 reading comprehension scene with inference-level questions — child reads between the lines
  • Harder than G2-3 (literal recall) — requires synthesizing multiple clues into a conclusion
  • On correct: evidence explanation box shown — teaches reasoning, not just confirms "right"
  • Coral theme. No timer, no accuracy %, stars never decrease.

Tabs / States

  • Active Longer passage with subtle clue highlights, question type badge, 4 full-sentence choices
  • Correct Clues get green highlight, evidence explanation box appears below passage, choice → green, coral CTA
  • Wrong Wrong choice shakes, banner directs to specific clues in passage ("slow walk, eyes down…")
  • Spec This document

Passage

  • 5–7 sentences, 0.88rem, line-height 1.65
  • Clue sentences have .clue class → subtle coral background (not obvious — hints, not gives away answer)
  • On correct: .correct-hl added → green highlight to show evidence

Question Type Badge

  • Small badge above question: "🔍 Inference" / "📖 Vocabulary" / "🗺️ Main Idea" / "⚡ Cause & Effect"
  • Teaches metacognitive reading skill labels — G4-5 students beginning to learn skill names

Evidence Explanation (Correct Only)

  • Green box with "📌 Evidence" label + explanation of WHY the answer is correct
  • Uses textual evidence vocabulary: "This shows…", "Clues include…", "We can infer because…"
  • This is the KEY difference from G2-3 scene — not just "correct!" but explains reasoning

Question Types (G4–5)

  • Inference: what the character feels/thinks, not explicitly stated
  • Vocabulary in context: "What does 'reluctantly' most likely mean in paragraph 2?"
  • Main idea: "What is the passage mainly about?"
  • Cause and effect: "Why did Priya sit quietly beside Dani?"

Design Tokens (G4–5 Coral Theme)

  • Coral: #ff7b6b — progress segs, passage title, clue highlight, q-type badge, CTA
  • Clue active: rgba(255,123,107,0.15) bg
  • Clue correct: rgba(80,232,144,0.2) bg
  • Evidence box: #0d2a1a bg / #50e890 border + text
  • CTA: linear-gradient(135deg, #ff7b6b, #d04a38), white text
