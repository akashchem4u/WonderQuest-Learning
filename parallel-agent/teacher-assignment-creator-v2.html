created_at      TIMESTAMPTZ DEFAULT now()
Privacy Rules
Scope Teacher sees class-level aggregates only; never individual student breakdowns.
Accuracy % No per-student accuracy percentage is surfaced anywhere in this component.
Student names Students identified by first name only for engagement counts; they are never listed by name in this view.
FERPA Assignment data is school-linked. Not personally disclosed externally. Compliant with 20 U.S.C. § 1232g.
Business Rules
BR-1 A teacher may only create assignments for classes they are the assigned teacher of (teacher_id must match the class record).
BR-2 If assignment_type is 'quest_batch', quest_batch_id must be provided and must belong to an active, published batch.
BR-3 If assignment_type is 'skill_area', skill_area must be one of: Reading, Maths, Science, Social Studies, Arts.
BR-4 If assigned_to is 'group', group_ids must be a non-empty array of valid group IDs belonging to the class.
BR-5 due_date, when provided, must be a future date (≥ tomorrow in the teacher's local timezone).
BR-6 teacher_note is optional but must not exceed 200 characters. Profanity filtering is applied server-side before persistence.
