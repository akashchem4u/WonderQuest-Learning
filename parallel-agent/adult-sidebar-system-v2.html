adult-sidebar-system-v2.html — Desktop sidebar navigation system. Used by: Teacher (blue accent), Parent (violet accent), Owner (green accent). Layout: fixed 240px wide sidebar + flex:1 content area (demo only). Sidebar variants: Expanded (240px, icon+label), Collapsed (60px, icon only, tooltip on hover). Active item: coloured left border (4px) + subtle bg highlight. Badges: notification count bubble on nav items. Collapse trigger: chevron button at bottom of sidebar. Storage: sidebar_state saved to localStorage('wq_sidebar_collapsed'). Persona switching: sidebar header changes logo tint + nav items entirely.

Nav Item Data Model
sidebar_nav_items(
  id            UUID PRIMARY KEY,
  persona       ENUM('teacher', 'parent', 'owner'),
  section       TEXT,             -- e.g. 'Main', 'Classroom', 'Tools'
  label         TEXT,             -- Display label
  icon          TEXT,             -- Emoji or icon key
  route_path    TEXT,             -- e.g. '/teacher/home'
  sort_order    INT,
  badge_key     TEXT NULLABLE,    -- key to look up in live counts
  requires_role TEXT[]            -- RBAC: e.g. ['teacher', 'admin']
)
Badge Count Sources
Badge Key Source Query
review_queue SELECT COUNT(*) FROM review_queue WHERE assigned_to = :user_id AND status='open'
interventions SELECT COUNT(*) FROM interventions WHERE class_id = :class_id AND status='active'
messages SELECT COUNT(*) FROM messages WHERE recipient_id = :user_id AND read_at IS NULL
feedback_queue SELECT COUNT(*) FROM feedback WHERE status IN ('open','investigating') AND assigned_to IS NULL
p0_alerts SELECT COUNT(*) FROM platform_incidents WHERE severity='P0' AND resolved_at IS NULL
Layout Rules
  • Sidebar: position:fixed; top:0; left:0; height:100vh; overflow-y:auto; z-index:100
  • Expanded width: 240px — Collapsed width: 60px
  • Width transition: width 200ms ease
  • Main content offset: margin-left:240px (expanded) / margin-left:60px (collapsed)
  • Persistence: localStorage('wq_sidebar_collapsed') = 'true' / 'false'
  • Scrollbar: .sidebar::-webkit-scrollbar { width: 4px } thin custom track

Collapsed State (60px)
  • Icon only — .nav-label opacity:0, width:0 (CSS transition)
  • Section labels hidden: opacity:0; height:0
  • Badge count hidden; replaced by 8px red dot at top-right of icon
  • Tooltip on hover via CSS ::after or sibling element, left: calc(100% + 10px)
  • Collapse button shows › (pointing right) to expand
  • Header shows WQ logo circle only — wordmark hidden
  • Footer plan chip and version label: opacity:0; height:0; overflow:hidden
