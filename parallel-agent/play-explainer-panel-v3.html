Trigger Logic

  • ✅ Show full panel on first-ever encounter of a question type
  • ✅ Show abbreviated panel on encounters 2–3
  • ✅ Stop showing entirely after encounter 4+
  • ✅ Track per child × per question-type (not global)
  • ❌ Never show explainer mid-session (only before question 1 of a type)
  • ❌ Never auto-advance — child must tap CTA or Skip

Layout Rules

  • Full sheet: ~55–65% of phone frame height
  • Abbreviated sheet: ~22–28% of phone frame height
  • Always has drag handle (::before 36px bar)
  • Content above sheet is dimmed with scrim overlay
  • Full sheet has Skip link; abbreviated sheet has no Skip
  • Sheet always position: absolute; bottom: 0

Band Color Tokens

  • Pre-K: gold #ffd166 border-top, dark #1e1400 bg
  • K-1: violet #9b72ff border-top, dark #1a1040 bg
  • G2-3: mint #58e8c1 border-top, dark #0a1e1a bg
  • G4-5: coral #ff7b6b border-top, dark #1e0e0a bg
  • Step number circles: filled with band color
  • CTA button: linear-gradient from band color → darker shade

Content Guidelines

  • Pre-K: ≤2-word steps (Look / Find / Tap)
  • K-1: short action sentences, sound-it-out language
  • G2-3: math framing ("groups of"), skip-count hint
  • G4-5: inference framing, "detective thinking" metaphor
  • ❌ Never say "If you get it wrong…"
  • ❌ Never mention scoring/accuracy in panel

Abbreviated Panel Rules

  • Single row: icon + 1 sentence (≤12 words)
  • No steps, no example card
  • Scrim opacity: 0.30 (lighter than full = 0.55)
  • "Got it!" → no "Let's go" suffix (brevity)
  • No Skip link (short enough to not need escape)
  • Same band color border-top and CTA gradient

Accessibility / UX

  • CTA button: min height 52px (touch target)
  • Skip link: min tap area 44px (add padding)
  • Sheet: border-radius: 24px 24px 0 0
  • Drag handle: cosmetic only — no swipe-dismiss in v3
  • No animation on sheet entry in this file (shown static)
  • Sheet slide-up: translateY(100%) → 0, 0.3s ease-out
