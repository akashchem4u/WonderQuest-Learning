Engineering Notes — child-returning-player-hub-v2.html

  • 5 states in one file — JS showTab() swaps both tab buttons and phone screen visibility
  • Confetti fires via burstConfetti() on 30-day return CTA click — 16 dots, CSS --tx/--ty vars
  • Active node uses node-pulse box-shadow animation (same-day tab)
  • Gap-day bar uses dashed border for missed days, solid violet for today, solid green for played
  • Star-safe badge shown on all gap screens (2d, 7d, 30d) — not on same-day
  • Streak dots: 7 dots, lit dots have gold glow — shows partial week progress
  • Streak restore (7d tab): emoji appears with restore-appear bounce keyframe
  • 30-day return: gold gradient on CTA, legend badge chip, trophy mascot enlarged to 88px
  • Threshold logic lives in backend — frontend receives returnType param to determine which state to show
  • World progress unchanged across ALL gap durations — crystal caverns still at 58% / node 7
