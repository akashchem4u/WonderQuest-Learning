Rarity System

  • Common (⭐): session completion, first correct; unlimited
  • Rare (⭐⭐): 5-streak, 3-session streak; limited 9999
  • Epic (⭐⭐⭐ rare): skill mastery, boss 3×; limited 4999
  • Legendary (⭐⭐⭐ gold): special achievements; limited 999
  • Each badge has a serial number (#042 / 999)
  • Glow intensity increases with rarity

Badge Flip Animation

  • Container: perspective: 600px
  • Keyframe: rotateY(90deg) scale(0.6) → rotateY(-8deg) scale(1.05) → rotateY(0) scale(1)
  • Duration: 0.7s, spring cubic-bezier(0.34,1.56,0.64,1)
  • Delay: 0.15s (after announce banner fades in)
  • Glow: badge-specific CSS box-shadow, rarity-colored

Flow

  • Reveal → tap → Detail → tap → Gallery OR dismiss
  • Skip link on Reveal → goes directly to Gallery
  • Gallery → tap badge → Detail
  • No auto-dismiss on any step
  • Child must tap to advance
  • Bonus XP is awarded immediately on badge earn (not on tap)

Gallery

  • Grid: 3 columns, 1:1 aspect ratio cells
  • Locked badges: opacity 0.4, "???" name, gray emoji
  • New badge: gold border + floating ✨ pip
  • Legendary bg: gradient dark gold
  • Tapping locked badge: no action (or "Keep playing to unlock!")
  • Max visible: 9 in preview (3×3) — paginated scroll for more
