ENG NOTE — teacher-class-health-mini-board-v2.html
Class health mini board — compact at-a-glance widget showing 4 class health dimensions on teacher home.
Dimensions: Engagement (sessions/active students this week) | Mastery velocity (skills mastered vs last week) | Queue load (open support queue items) | Streak health (students maintaining active streaks).
Colour rules: Green = healthy/good. Amber = attention needed (not alarming). Blue = informational/positive. Red = NEVER used here (red reserved for system errors only).
Deltas: Week-over-week change. Down deltas always amber (never red).
Placement: Embeddable panel on teacher home right column (below recommendations), or expanded card on Command Center / Class Overview.
Not a deficit view: All language is neutral/supportive. "4 students in queue" not "4 struggling students".
