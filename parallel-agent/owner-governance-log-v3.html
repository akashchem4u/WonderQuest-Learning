Component: owner-governance-log-v3.html
Immutable audit trail. All significant actions logged. 7-year FERPA retention. Filter by category. Export CSV/PDF.

Log Categories

Privacy: Data deletion, anonymisation, consent changes, FERPA/GDPR actions. Content: Skill publish/block/unblock, content review decisions. Ops: Incidents, deployments, system events. Security: MFA changes, login anomalies, access changes. Release: Version launches, hotfixes. Account: School onboard/offboard, DPA changes.

Immutability

Log entries are append-only. No update or delete operations on log_entries table. Write access is restricted to system service account only. Owner can view + export but cannot modify. Regular integrity checks compare log hash chain.

Retention

7-year retention per FERPA requirements (student education records). Privacy/FERPA entries are kept for the full 7 years even after school contract ends. Ops/Release entries purged after 3 years. Security entries kept 5 years.

Actor Field

"System (auto)" for automated actions (monitoring, cron jobs, auto-block). "Owner" for manual owner actions. Engineering team name for deploy actions. No individual teacher names — school name only for school-initiated events.
