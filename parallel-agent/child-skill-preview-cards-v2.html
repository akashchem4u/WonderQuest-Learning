Component

File
child-skill-preview-cards-v2.html
Usage
Shown before quest starts, on quest selector detail, and in /child/skills view
Layout
Grid: auto-fill minmax(240px,1fr). Detail: 2-col (1fr + 300px related panel)

Card States

New/available
Default border + card-pop entry animation
Active next
violet border + darker bg (#1e1470)
done-sc
Green border (#50e890) + progress bar at 100% or partial
locked-sc
50% opacity · "Finish X first" unlock message

Language Rules

Skill names
Child-friendly action phrases ("Find the Word!" not "Word Recognition")
No grade labels
"K–1 Band" never "Grade 1". No skill IDs in child view.
No accuracy %
Stars only. Never "you got X% correct".
Done desc
"Already mastered this one! Replay anytime." — encouragement not metrics

Star Safety

Replay
All completed skills replayable. Stars never decrease on replay.
Badge
"⭐ Stars are always safe — replay anytime!" on related panel

Animation

Entry
card-pop: opacity+scale(0.92)+translateY(8px) → normal · 0.4s spring
Stagger
nth-child delays 0.05s per card (up to 6 cards)
