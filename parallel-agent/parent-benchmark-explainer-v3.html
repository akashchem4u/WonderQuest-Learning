ℹ️ How WonderQuest works
Understanding Maya's progress
WonderQuest uses learning bands, mastery scores, and stars to show how Maya is developing — not percentages or grades. Here's what everything means.
The 4 learning bands
Pre-K Band
Ages 4–5
Letter recognition, phonics foundations, counting to 20
K–1 Band
Ages 5–7
Sight words, blending, addition facts, basic spelling
Grades 2–3
Ages 7–9
Fluency, multi-digit math, vocabulary expansion
Grades 4–5
Ages 9–11
Comprehension, fractions, complex spelling patterns
Skill status badges
⭐ Strong Maya has a solid grasp of this skill. Keep going — higher levels await!
🌱 Building This skill is actively developing. Normal and expected — a few more sessions will build it up.
🌱 Just started Brand new skill — Maya has only seen it a few times. Early days!
How the mastery bar works
1
Maya answers questions on a skillWonderQuest tracks how consistently she gets them right over multiple sessions — not just one.
2
The mastery bar fills upIt rises when she consistently succeeds, and holds steady (doesn't drop) when she gets a few wrong — because one bad day doesn't undo learning.
3
She earns stars along the wayStars reward effort and consistency — not perfection. This keeps learning fun.
Why doesn't Maya see percentages?
Research shows that showing accuracy percentages to young learners can reduce motivation and create anxiety. Stars celebrate effort. You see fuller data in the weekly report because context helps you understand and support — not judge.

Key Messages (must be consistent)

1. Mastery never decreases — one bad session doesn't undo learning.

2. Stars celebrate effort and consistency — not perfection or accuracy.

3. No accuracy % shown to child — intentional, research-backed. Parent sees fuller data in weekly report for context.

4. Band placement is not a grade — it's a starting point that adjusts as the child progresses.
