Desktop Layout

  • Frame: max-width 960px, border-radius 24px
  • Grid: grid-template-columns: 1fr 380px
  • Left hero min-height: 480px (right col matches)
  • Right summary bg: #0e0a28 (slightly darker)
  • Border between cols: 1px solid #1e1850
  • Session segs: 8px height (larger than mobile 6px)

Star-Safe Rules

  • Stars NEVER decrease on any action on this screen
  • "Play Again" triggers star-safe overlay before replay
  • Star counter shows POST-session total (already credited)
  • ❌ Never say "You got X/5 correct" — stars only
  • ❌ Never show accuracy % anywhere on this screen
  • ❌ Never show wrong-answer count in summary

Summary Card Rules

  • Stats: Questions (always "5 of 5"), Stars, XP, Streak
  • No "wrong answers" stat ever
  • Perfect bonus: separate highlighted stat row
  • Badge earned: separate highlighted stat row (gold tint)
  • Coach: band-specific, positive, forward-looking
  • CTA changes when badge earned: "See Your Badge →" primary

Band Variants

  • Band chip: Pre-K gold, K-1 violet, G2-3 mint, G4-5 coral
  • Segs fill: band accent (gold on perfect session always)
  • Primary CTA gradient: band-specific
  • Perfect session: primary CTA always gold regardless of band
  • Badge earned CTA: always gold (badge is special)
