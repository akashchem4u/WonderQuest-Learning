Data Shown (Child View)

  • Stars earned per session (always positive delta)
  • Skill name + band
  • Relative date ("Today", "Yesterday", "2 days ago")
  • Perfect session pip (gold)
  • Badges earned in that session
  • ❌ NEVER show wrong-answer count in history

Privacy Rules (Child History)

  • This is the CHILD's view — celebratory, no negatives
  • Parent's detailed report is separate (parent-weekly-report-v3.html)
  • Child never sees "X wrong answers" or accuracy %
  • Child never sees skill names they struggled with prominently
  • All-time star total shown as motivator
  • ❌ Never cross-contaminate parent data into child view

Strip Layout

  • Card width: 130px; gap: 10px
  • Horizontal scroll: overflow-x: auto; scrollbar-width: none
  • Cards: flex-shrink: 0
  • Scroll snapping: optional (smooth feels better)
  • Max visible: 3 cards before scroll; show partial 4th as hint

Expanded Detail

  • Tap card → expand inline (not a new screen)
  • Close via × button → collapses back to strip
  • 5 segs shown filled (all done)
  • Stats: Stars, XP, Questions count
  • "Play again" → launches new session with same skill
  • Other cards remain visible below (scrollable)
